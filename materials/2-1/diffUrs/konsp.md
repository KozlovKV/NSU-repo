# 23.09.07 - лекция
## Основные понятия
Мы будем говорить только об обыкновенны дифференциальных уравнения (*БОЛЬШОЕ СПАСИБО*)

**О.** Обыкновенное дифф. уравнение порядка $n$ относительно неизвестной функции $y(x)$ - уравнение вида $F(x, y, y', ..., y^{(n)}) = 0$. В дифф.уре всегда есть производные. Порядком уравнения называется порядок старшей производной. Разделение диффуров на классы производится по их порядку. Для разных классов есть разные методы решения.

**О.** Решение диффура - это функция $y = \phi(x)$, если при подстановке в уравнение $y \rArr \phi, y' \rArr \phi'$ получается тождество. Как правило уравнение имеет бесконечное количество решений, которые, также, зависят от произвольных констант C, такое решение записывается как $\phi(x, C_1, C_2, ..., C_n)$ - такая запись называется **общим решением диффура**

Часто в ходе преобразований мы получаем лишь **нефявное задание** - функциональное соотношение вида $\Phi(x, y, C_1, C_2, ..., C_n) = 0$. Обычно перейти к явному решению от неявного мы можем лишь локально

**Нормальная форма** - форма диффура, в которой мы старшую производную выражаем через остальные производные: $y^{(n)} = f(x, y, y', ..., y^{(n-1)})$

**О.** График решения диффура $y = \phi(x)$ называется **интегральной кривой**

**Поле направлений** для системы $y' = f(x, y)$ - множество всех прямых, проходящих через точки $(x, y)$ ООФ под углом, тангенс которого будет равняться $f(x, y)$. Для интегральной кривой диффура поле её касательных совпадает с полем направлений этого уравнения.

*Не уверен, что важно хоть где-то, но пусть будет:* для уравнений второго порядка всякая интегральная кривая определяется связью между координатами точек этой кривой, наклоном касательной и кривизной в каждой её точке

**Для системы диффуров большинство определений аналогичны**

Связь интегральных кривых с полем направлений и возможность представить их в параметрическом виде позволяет представить нам в параметрическом виде и решение диффура (то есть $x = \phi(t)$, $y = \psi(t)$)

**Система обыкновенных дифференциальных уравнений** - совокупность ОДУ, в каждом из которых есть $y_1, y_1', ..., y_1^{m_1}, ..., y_n, y_n', ..., y_n^{m_n}$. Соответственно, решением будет $n$ функций, обращающих систему в тождество. *Нормальная форма и общее решение также определяются аналогично*

**Интеграл системы диффуров первого порядка в нормальной форме** в окрестности точки $(x_0, y_{01}, ... y_{0n})$ - функция вида $\phi(x, y_1, ..., y_n)$, которая в этой точке имеет непрерывные первые частные производные и при этом сумма частных производных равняется нулю (частные производные по $y_i$ домножаются на $f_i(x, y_1, ..., y_n)$)

**Общим интегралом системы** будет система, в которой от равенств $\phi_i(x, y_1, ..., y_n) = C_i$ мы можем перейти к $y_i = \psi_i(x, C_1, ..., C_n)$

**Задача Коши для заданного ОДУ** состоит в том, чтобы из всех решений найти такое решение $y = y(x)$, которое вместе со своими производными $n - 1$ порядка принимает заданные значения $y_0, y_1, ..., y_{n-1}$ при заданном значении независимой переменной $x_0$. Начальными значениями будут называться заданные $x_0, y_0, ..., y_{n-1}$. *Аналогичным образом можно определить задачу Коши для системы ОДУ первого порядка (с индексами только чутка подшаманить)*

**Траектория ОДУ первого порядка** - $(\phi_1(t), \phi_2(t), ..., \phi_n(t))$ - при изменении параметра $t$ эта точка будет описывать кривую $n$-мерном пространстве. Каждая координата точки задаётся решением ОДУ: $\phi_1(t) = y_1, \phi_2(t) = y_2, ..., \phi_n(t) = y_n$. Соответственно, для систему ОДУ задача Коши сводится к поиску подходящей траектории

**Не у всякой задачи Коши есть решение и не у всякой оно единственное.** Решение, в котором в каждой точки нарушается условие единственности, называется **особым**.

*Пример:* Для $y' = \sqrt{y}$ и задачи Коши $y(x_0) = 0$ будет 2 решения: $y(x) = \frac{1}{4}(x - x_0)^2$ и $y(x) \equiv 0$ (*особым решением тут будет $y(x) \equiv 0$*)

### Сведение к системе ОДУ
ОДУ порядка $n$ в нормальной форме может быть преобразовано в систему ОДУ первого порядка путём не очень хитрого переобозначения: $y \harr y_1, y' \harr y_2, ..., y^{(n-1)} \harr y_n \rArr y_{(n-1)}' = y_n$. Считая, что исходное ОДУ записано в нормальной форме, получаем из него систему:
$$
\begin{cases}
  y_1' = y_2 \newline
  y_2' = y_3 \newline
  ... \newline
  y_{(n-1)}' = y_n \newline
  y_n' = f(x, y_1, y_2, ..., y_n)
\end{cases}
$$

Возможен и обратный процесс, то есть переход от системы из $n$ ОДУ в НФ первого порядка к одному ОДУ $n$-го порядка. Для этого требуется существование первых частных производных для функции одного из уравнений системы, а также возможность выразить одну незивестную функцию через другую (то есть $y_2 = g_1(x, y_1, y_1')$ и наоборот).

## Простейшие классы диффуров и методы их решения

Хотя при $f_1 \ne 0$ мы можем перейти от второго вида к первому, это сузит множество точек $(x, y)$, поэтому есть риск потерять решение. В таком классе диффуров возможно выразить $x$ как функцию от $y$ и $C$ и решать диффур $x' = \frac{1}{f(x, y)}$ либо $f_1(x, y) = f_2(x, y)x'$, что в некоторых случаях будет удобнее. Если переменные находятся под одной и той же функцией и не могут разделены, то мы производим замену выражения под функцией. Пример: уравнение $y' = f(ax + by + c)$. Берём $z = ax + by + c$, считаем $dz = adx + bdy \rArr dy = \frac{dz - adx}{b}$, подставляем в исходное уравнение:
$$
\frac{dz - adx}{bdx} = f(z) \hArr \frac{dz}{bdx} - \frac{a}{b} = f(z) \hArr z' = a + bf(z)
$$
Далее решаем диффур по обычным правилам.

К однородным диффурам можно привести следующие диффуры:

y' = f(\frac{a_1x + b_1y + c_1}{a_2x + b_2y + c_2})

Здесь $f$ - однородная функция степени 0

$$

\begin{cases}

x_0 = 1, y_0 = 2 \rArr x = t + 1, y = z + 2 \newline

## Линейные дифференциальные уравнения
**О.** Диффуры первого порядка вида $y' + p(x)y = q(x)$ называются **линейными диффурами первого порядка**. $p(x)$ - **коэффициент диффура**, $q(x)$ - его правая часть

Метод решения (**метод вариации постоянной** или **метод Лагранжа**):
1. Решаем уравнение, где рассматриваем $q(x) = 0$. В таком случае у нас будет простое однородное уравнение. Берём полученный ответ вида $y = C\phi(x)$
2. Для $q(x) \not= 0$
   1. Делаем подстановку решения однородного уравнения при $q(x) = 0$, заменяя константу на неизвестную $y = C(x)\phi(x)$
   2. Поиск решения будет сводиться к поиску $C(x)$ классическими методами

Запись линейного диффура в симметричной форме: $dy - [f(x)y + g(x)]dx = 0$ (*ИМХО. Очень неудобная и некрасиваф форма, которая ещё и требует преобразований для использования метода Лагранжа*)

Некоторые диффуры можно привести к линейному виду, представив $x(y)$, то есть поменять местами зависимую и независимую переменные (*ясное дело, мы не просто меняем букву производной, а проводим преобразования, держа в голове, что $y'^{-1} = x'$*) $(2e^y - x)y' = 1 | / y' \rArr 2e^y - x = x' \rArr x' + x = 2e^y$

Уравнение вида $f'(y)y' + a(x)f(y) = b(x)$ также приводится к линейному заменой $z = f(y) \rArr z' + a(x)z = b(x)$

*Пример:*
$$
\frac{y}{\sqrt{y^2 + 1}}y' + \sqrt{y^2 + 1} = x^2 + 1 \newline
z = \sqrt{y^2 + 1} \rArr z' = \frac{yy'}{\sqrt{y^2 + 1}} \newline
z' + z = x^2 + 1
$$

Уравнение вида $y' + p(x) = q(x)e^{ny}, n \not = 0$ сводится к линейному заменой $z(x) = e^{-ny(x)} \rArr z'(x) = -ne^{-ny(x)}y'(x) = -nz(x)y'(x)$. Тривиальными преобразования получаем $\frac{z'}{-n} + p(x)z = q(x)$

### Уравнение Бернулли
**Уравнение Бернулли** - диффур вида $y' + p(x)y = q(x)y^a$ при $a \not \in {0, 1}$ (*при этих значения мы получаем простой линейный диффур*). Для перехода к линейному диффуру производим замену $z(x) = y^{1 - a}(x) \rArr z'(x) = (1 - a)y^{1 - a - 1}(x)y'(x)$: $z' + (1 - a)p(x)z = (1 - a)q(x)$.

Также при $a > 0$ **дополнительным решением уравнения Бернулли** будет $y = 0$

### Уравнение Рикатти
**Уравнение Рикатти** - диффур вида $y' + a(x)y + b(x)y^2 = c(x)$. В общем случае не решается в элементарных функциях, но если мы знаем частное решение $y = y_1(x)$, то можем провести замену $y = y_1 + z$, которая сведёт уравнение Рикатти к уравнению Бернулли, которое сведётся к линейному заменой $u = \frac{1}{z}$

## Уравнение в полных дифференциалах
**Уравнение в полных дифференциалах** - уравнение вида $M(x, y)dx + N(x, y)dy = 0$, то есть его левая часть является полным дифференциалом некоторой непрерывно дифференцируемой $F(x, y)$

Для таких уравнений имеет место равенство: 
$$
dF = \frac{\delta F}{\delta x}dx + \frac{\delta F}{\delta y}dy
$$
А также должно выполняться равенство:
$$
dF = M(x, y)dx + N(x, y)dy
$$
Отсюда следует, что:
$$
\frac{\delta F}{\delta x} = M(x, y), \space\space\space \frac{\delta F}{\delta y} = N(x, y)
$$
Для всех этих равенств необходимое и достаточное условие:
$$
\frac{\delta M(x, y)}{\delta y} = \frac{\delta N(x, y)}{\delta x}
$$
Если верно последнее, то решаться диффур будет через последовательное интегрирование:
$$
\frac{\delta F}{\delta x}(x, y) = M(x, y)
$$
Интегрируем это равенство по переменной $x$, считая $y$ константой:
$$
F(x, y) = \int M(x, y) + \phi(y) = Ф(x, y) + \phi(y)
$$
$\phi(y)$ - произвольное слагаемое, зависящее от $y$, по ней и будет дифференцировать в обратную сторону
$$
\phi'(y) = (F(x, y) - Ф(x, y))' = N(x, y) - \frac{\delta Ф}{\delta y}
$$
**Правая часть здесь обязательно будет зависеть только от $y$**. Интегрируем это уравнение и находим $\phi(y)$ и находим подстановкой в $F(x, y) = Ф(x, y) + \phi(y)$

*Человеческая интерпретация:*
1. У нас есть утверждения о частных производных вида $\frac{\delta F}{\delta x} = M(x, y)$ и $\frac{\delta F}{\delta y} = N(x, y)$
2. Проводим интегрирование одной из них по той переменной, от которой взять частный дифференциал, другую переменную обозначаем как зависимую "константу"
3. Вычислив таким образом частичную форму $F(x, y)$ с зависимой "константой", подставляем её слева во второй частную производную из пункта 1 и считаем его
4. В итоге нам остаётся решить тривиальное дифференциальное уравнение, в котором будет только $y$

**Не всякое уравнение вида $M(x, y)dx + N(x, y)dy = 0$ является уравнением в полных дифференциалах**, то есть не все удовлетворяют условию $\frac{\delta M(x, y)}{\delta y} = \frac{\delta N(x, y)}{\delta x}$, однако почти всегда существует такая функция $\mu(x,y)$, что при умножении на неё $M,N$ уравнение придёт к этому виду, то есть выполнится условие:
$$
\frac{\delta}{\delta y}(\mu(x, y)M(x, y)) = \frac{\delta}{\delta x}(\mu(x, y)N(x, y))
$$
В таком случае $\mu(x,y)$ - **интегрирующий множитель**. Если он известен и не обращается в нуль, то диффур решается по изложенной выше схеме

Для поиска $\mu$ надо вычислить уравнение:
$$
\frac{\delta (\mu M)}{\delta y} = \frac{\delta (\mu N)}{\delta x} \newline
\frac{\delta \mu}{\delta y}M + \frac{\delta M}{\delta y}\mu = \frac{\delta \mu}{\delta x}N + \frac{\delta N}{\delta x}\mu
$$
Делим на $\mu$ и, где она останется, вносим её под дифференциал:
$$
\frac{\delta \ln\mu}{\delta y}M + \frac{\delta M}{\delta y} = \frac{\delta \ln\mu}{\delta x}N + \frac{\delta N}{\delta x}
$$
Подставляем значения $M, N, \frac{\delta M}{\delta y}$ и $\frac{\delta N}{\delta x}$, а далее находим решения, зависящие только от одной переменная (*убираем слагаемое с частным дифференциалом от логарифма по другой, считая его равным нулю*) и таким образом находим $\frac{\delta \ln\mu}{\delta y}$ либо $\frac{\delta \ln\mu}{\delta x}$, а далее вычисляем значение $\mu$, интегрируя по нужной переменной

## Уравнения первого порядка, неразрешимые относительно производной
Общий вид уравнений первого порядка неразрешённых относительно производной таков: $F(x, y, y') = 0$. 

В простейшем случае левая часть уравнение элементарно представляется в разрешённом относительно производной $F(x, y, y') = (y' - f_1(x,y))(y' - f_2(x,y))...(y' - f_m(x,y))$, при этом диффур распадается на $m$ диффуров вида $y' = f_1(x,y), ..., y' = f_m(x,y)$ - решение каждого из них будет решением исходного диффура, то есть будет ветвью общего решения. Кроме того, решениями будут частные особые решения этих систем, а также решения, полученные склеиванием нескольких ветвей решения.

В самом простом случае у нас будет диффур $F(y') = 0$, в котором функциональное уравнение $F(\xi) = 0$ имеет вещественные корни $\xi = a_1, ..., \xi = a_n$, тогда решения диффура будут неявно задаваться формулой $F(\frac{y - C}{x}) = 0$ (*получаем такое выражение, просто интегрируя $y' = a_i$*), однако если решения $\xi$ заполняют некоторый интервал, эта формула может не давать всех решений диффура

Для диффура $F(x, y') = 0$, в таком случае мы можем найти функции для уравнения $F(\xi, f_i(\xi)) = 0$, а решения диффура тогда будут находиться интегрированием $f_i$-х

## Параметрическое решение дифференциальных уравнений
Диффур $F(x, y, y') = 0$ Допускает запись в параметрической форме, если $\exist \phi(u,v), \psi(u,v), \chi(u,v)$ параметров $u,v$ такие, что имеет место тождество $F(\phi(u,v), \psi(u,v), \chi(u,v)) \equiv 0$

Из записи, справедливой для любого решения диффура - $dy = y'dx$ - получаем через подстановки следующую:
$$
\frac{d\psi}{du}du + \frac{d\psi}{dv}dv = \chi(u, v)(\frac{d\phi}{du}du + \frac{d\phi}{dv}dv)
$$
Относительно переменных $(u,v)$ мы получаем обыкновенный диффур в симметричном виде.

Если решение было найдено в форме $v = w(u, C)$, то мы получаем:
- $x = \phi(u, w(u, C))$
- $y = \psi(u, w(u, C))$

*Очевидным образом эти 2 записи преобразуются, если у нас было найдено $u = w(v, C)$*

*Практическое применение метода затрудняется поиском этих самых параметрических функций*

Однако есть 2 случая, в которых поиск параметрических функций сильно упрощён:
- Если $F(x, y, y') = 0$ разрешим относительно $y$: $y = \psi(x, y') \rArr \phi = u = x, \chi = v = y'$
- Если $F(x, y, y') = 0$ разрешим относительно $x$: $x = \phi(y, y') \rArr \psi = u = y, \chi = v = y'$
- *В обоих этих случаях сильно упрощается запись выражения $dy = y'dx$, так что приводить её тут не буду, смогу если что вывести*

## Продолжаем про неразрешимый первый порядок
**Уравнение Лагранжа** - диффур вида $y = x \phi(y') + \psi(y')$, в котором $\phi(\xi) \not \equiv \xi$. Уравнение разрешимо относительно $y \rArr y = \psi(u, v), x = u, y' = v$, Затем запишем уравнение в дифференциалах $du, dv$ и далее действуем классическим параметрическим методом

Если в уравнении Лагранжа $\phi(\xi) \equiv \xi$, то мы получаем **уравнение Клеро**. Выражаем его через те же параметры и получаем $[u + \psi'(v)]dv = 0 \rArr dv = 0$ или $u + \psi'(v) = 0$. Первое уравнение имеет решение $v = C \rArr y = Cx + \psi(C)$. Второе уравнение имеет особое решение уравнения Клеро $x = -\psi'(v), y = -v\psi'(v) + \psi(v)$ - в каждой точке соответствующей ему интегральной кривой нарушается единственность решения задачи Коши. 

## Решение диффуров высокого порядка
[Про простые способы понижения порядка смотри в семинаре](#231010---семинар)

При решении задачи Коши для диффуров выского порядка целесообразно находить значения константу сразу же после их появления, а не всех в конце (*вообще слабо представляю, как найти значения всех в конце*)

Диффур вида $y^{(n)} = 0$ будет решаться последовательным интегрированием $n$ раз. Ответ будет выглядеть так: $y = a_1x^{n-1} + a_2x^{n-2} + ... + a_{n-1}x + a_n$

Диффур порядка $n$ в нормальной форме сводится к системе диффуров первого порядка (смотри [тут](#сведение-к-системе-оду))

### Линейный диффур высокого порядка
$$
y^{(n)} + a_1(x)y^{(n-1)} + a_2(x)y^{(n-2)} + ... + a_{n-1}(x)y' + a_n(x)y = f(x)
$$
**Линейное уравнение порядка $n$**. Также может записываться в **операторной форме**
$$
\frac{d^n y}{dx^n} + a_1(x)\frac{d^{n-1} y}{dx^{n-1}} + ... a_{n-1}(x)\frac{dy}{dx} + a_n(x)y = f(x)
$$
Если в этой форме $a_i$ - константы, то говорят **линейном уравнении с постоянными коэффициентами**

При $f(x) = 0$ ЛДУ будет **однородным**, иначе - **неоднородным**

*Также существует ещё одна максимально извращённая запись с подстановкой $D = \frac{d}{dx}$:*
$$
(D^n + a_1D^{n-1} + .... + a_{n-1}D + a_n)y = f(x)
$$

**Характеристический полином** $p(\lambda) = \lambda^n + a_1 \lambda^{n-1} + ... + a_{n-1}\lambda + a_n$ будет характеристическим полиномом линейного однородного уравнения, если $a_i$ - постоянные уравнения, равны $a_j$ полинома

Уравнение $p(\lambda) = 0$ будет называться **характеристическим** для данного диффура, причём его корням будут вещественные числа 

Если полином $p(\lambda)$ имеет $k$ вещественных и комплексных корней (для условия вещественности каждому комплексному корни будет соответствовать комплексно сопряжённый), то, оказывается соответствующий ему диффур в операторной форме можно записать в **факторизованной форме**
$$
(D - \lambda_1)(D - \lambda_2)...(D - \lambda_k)y = 0
$$
*Доказывается через пример для порядка `2` с использованием формулы Виета*

### Формула общего решения для линейных уравнений с постоянными коэффициентами
Для однородного линейного диффура в факторизованной форме порядка $n$ решениями будут $y_1 = C_1e^{\lambda_1x}, ..., y_n = C_ne^{\lambda_n x}$, а также любая их линейная комбинация. Если все $\lambda$ различны, то **линейная комбинация всех $y_i$** будет описывать **общее решение диффура**.

*Пример*:
$$
\frac{d^2 y}{dx^2} - \frac{dy}{dx} - 2y = 0 \newline
p(\lambda) = \lambda^2 - \lambda - 2 
\rArr \lambda_1 = -1, \lambda_2 = 2 \newline
(D + 1)(D - 2)y = 0 \rArr y = C_1e^{-x} + C_2e^{2x}
$$

Если у нас совпадают некоторые корни характеристического полинома $\lambda_1 = \lambda_2 = ... = \lambda_k = \lambda$, тогда вместо сумму в общем решении их слагаемые преобразуются в запись вида $(C_1 + C_2x + ... + C_kx^{k-1})e^{\lambda x}$

Для комплексно сопряжённых корней $\lambda_1 = a + bi, \lambda_2 = a - bi$ решением примет вид $e^{ax}[C_1\cos bx + C_2\sin bx]$, если таковых сопряжённых пар $k$ штук, получим $e^{ax}[(C_1 + C_2x + ... C_kx^{k-1})\cos bx + (B_0 + B_1x + ... + B_kx^{k-1})\sin bx]$

## Линейного неоднородного диффура высокого порядка
Общая форма диффура была описана выше. Алгоритм такой:
- Находим решение однородного диффура
- Находим любое частное решение неоднородного
- Складываем решения

Таким образом, самой сложной задачей становится поиск частного решения. Для этого существует несколько методов, схожих по своей сути (*использованию неопределённых коэффициентов*).

Если правая часть уравнения имеет вид $P_m(x)e^{ax}$ и при этом $a$ не будет корнем характеристического полинома, то частное решение будет иметь вид $Q_m(x)e^{ax}$. Подставляем частное решение в исходное уравнение и приравняем коэффициенты при одинаковых степенях $x$. Получим алгебраическую систему из $m + 1$ уравнений, благодаря которой найдём коэффициенты полинома $Q_m$:

*Пример:*
$$
\frac{d^2y}{dx^2} + 4y = 2x + 3 \newline
(D^2 + 4)y = 2x + 3 \newline
P(\lambda) = \lambda^2 + 4 \newline
\lambda = \plusmn 2i \newline
y_{общ} = C_1 e^{2ix} + C_2 e^{-2ix} \newline
y_{общ} = С_1 \cos 2x + C_2 \sin 2x \newline
2x + 3 \rArr Q_1(x)e^{0x} \newline
\rArr Q_1(x) = Ax + B \newline
4Ax + 4B = 2x + 3 \newline
\begin{cases}
  4A = 2 \newline
  4B = 3 \newline
\end{cases}
B = 3/4, A = 1/2 \rArr Q_1(x) = \frac{2x + 3}{4} \newline
\rArr y = С_1 \cos 2x + C_2 \sin 2x + \frac{2x + 3}{4}
$$

**Если $a$ - корень общего решения однородного диффура степени $k$**, то частное решение примет вид $x^kQ_m(x)e^{ax}$. В остальном метод решения идентичен

*Пример:*
$$
(D^3 + D^2)y = x^2 \newline
(D^3 + D^2)y = x^2e^{0x} \newline
P(l) = l^3 + l^2 \newline
l^3 + l^2 = 0 \rArr l^2(l + 1) = 0 \rArr l_1 = l_2 = 0 = a, l_3 = -1 \\
\rArr y_{общ} = C_1x + C_2 + C_3e^{-x} \newline
y_{частн} = x^2Q_2(x)e^{0x} = x^2(Ax^2 + Bx + C) = Ax^4 + Bx^3 + Cx^2 \newline
y^{(2)} = 12Ax^2 + 6Bx + 2C \newline
y^{(3)} = 24Ax + 6B \newline
24Ax + 6B + 12Ax^2 + 6Bx + 2C = x^2 \newline
\begin{cases}
  12A = 1 \newline
  24A + 6B = 0 \newline
  6B + 2C = 0 \newline
\end{cases}
\rArr A = 1/12 \rArr B = -1/3 \rArr C = 1 \newline
\rArr y_{частн} = \frac{x^4}{12} - \frac{x^3}{3} + x^2 \newline
\rArr y = C_1x + C_2 + C_3e^{-x} + \frac{x^4}{12} - \frac{x^3}{3} + x^2
$$

Если правая часть представлена в виде $e^{ax}[P_m(x)\cos bx + Q_l(x)\sin bx]$ и $a + bi$ **не является корнем общего решения однородного диффура**, то частное решение представима в виде $e^{ax}[R_p(x)\cos bx + T_p(x)\sin bx], p = \max(m,l)$. Соответственно, если $a + bi$ **является корнем степени $k$**, мы домножаем частное решение на $x^k$. В остальном метод решения идентичен.

**Если правая часть представима в виде суммы нескольких функций, то её частное решение будет суммой частных решений всех этих функций.** Бывает полезно в случаях, когда у нас смешаны вещественные и комплексные числа, или разные степени у экспоненты.

## Линейные уравнения с переменными коэффициентами
Вместо констант $a_i$ у нас теперь идут функции $a_i(x)$

$L(y) \equiv y^{(n)} + a_1(x)y^{(n-1)} + ... + a_{n-1}(x)y' + a_n(x)y$ - **линейный дифференциальный оператор порядка $n$** (*Именно так выглядит левая часть лин. диффура с переменными коэффициентами*). Линейный оператор обладает свойством линейности (*вот это поворот!*)

*Освежим память* функции $y_n(x)$ будут линейно независимы, если их линейная комбинация равняется нулю только при умножении функций на ноль ($\forall x \in <a,b> : \sum a_iy_i(x) = 0, a_1 = ... = a_n = 0$). Линейно независимыми будут системы $\{x^i\}$ и $\{e^{\lambda_ix}| \forall i\not=j : \lambda_i \not = \lambda_j \}$. Система $\cos^2 x, \sin^2 x, 1$ будет линейно зависима.

**Определитель Вронского** для системы из $n$ функций - определитель матрицы для системы из $n$ функций, где $w_{ij} = y_j^{(i-1)}(x)$ - производная j-й функции порядка $i-1$ (*то есть каждая строка - производные всех функций порядка от 0 до $n-1$*).

**Т.** Если система функций линейно зависима в интервале, то определитель Вронского будет равняться нулю в каждой точке этого интервала. *Доказывается через факт линейной зависимости и выражение функции $y_n$ через остальные функции, а затем её дифференцирование $n-1$ раз, чтобы получить последний столбец матрицы, который будет представлять собой линейную комбинацию предыдущих, из **чего и следует равенство определителя матрицы нулю***.

**Обратное теореме утверждение не верно** - не любая система функций с нулевым определителем Вронского будет линейно зависима.

**Т.** Пусть функции $y_i(x)$ линейно независимы на $(a, b)$ и каждая из них является решением диффура. $L(y) = 0$, тогда определитель Вронского для этих функций не будет равняться нулю ни в одной из точек $(a,b)$. *Доказывается от противного через предположение, что $y_i(x)$ - линейно независимые решения $L(y) = 0$ и при этом в какой-то из точек $(a, b)$ $W = 0$. Через решение СЛАУ и условие о единственности решения задачи Коши приходим к выводу, что система функций должна быть линейно зависимой $\rArr$ **доказано через противоречие***

**С. 1** Если для системы решений $L(y) = 0$ $\{y_i\}$ определитель Вронского равен нулю в одной точке, то он будет равен нулю во всех. *Доказывается выводом от противного прошлой теоремы и позапрошлой теоремой*

**С. 2** Если определитель Вронского системы решений $L(y) = 0$ $\{y_i\}$ отличен от нуля хотя бы в одной точке, то он отличен от нуля во всех точках. *Доказательство столь же тривиально, как у прошлого следствия*

**С. 3** Для линейной независимости системы решений $L(y) = 0$ $\{y_i\}$ **необходимо и достаточно**, чтобы определитель Вронского этой системы был отличен от нуля хотя бы в одной из точек области решения.

**Т. О формуле Остроградского-Лиувилля.** Если $y_i$ - решения $L(y) = 0$, тогда 

$$
\forall x,x_0 \in (a,b) : W(x) = W(x_0)e^{-\int_{x_0}^xa_1(s)ds}
$$
- **Существенно, что тут $a_1(x)$ именно по нашей нотации, то есть мы обязательно берём функцию при $y^{(n-1)}$, $a_0(x)$ строго $= 1$**
- Фактически, в практическом приложении мы считаем, что $W(x_0) = C$ (*что в целом и логично*)

*Доказывается через тот факт, что производная определителя Вронского будет равняться сумме $n$ определителей, где строки поочерёдно заменяются на свои производные (то есть при замене всех строк кроме последней определители будут нулевыми). Затем через умножение $n-1$ строчек определителя на $a_n, ..., a_2$ и факт того, что любое $y_i$ - решение, получаем (не особо понимаю, как), что $W'(x) = -a_1(x)W(x)$, а затем домножаем выражение на $e^{\int_{x_0}^xa_1(s)ds}$ и путём "нехитрых" преобразований получаем искомую формулу*

**Фундаментальная система решений** - совокупность решений $L(y) = 0$, определённых и линейно независимых на $(a, b)$.

**Т.** Для любой $L(y)$ с непрерывными на $(a,b)$ коэффициентами $a_i(x)$ найдётся ФСР. *Доказывается подбором таких решений $y_1(x), ..., y_n(x)$, что их определитель Вронского будет определителем диагональной матрицы, то есть будет равняться единице, а из этого следует, что система из таких решений линейно независима*

**Т.** Любая линейная комбинация ФСР также будет решением $L(y) = 0$ (*На самом деле формулировка и теоремы другая, но так это мне кажется вполне корректным*)

Также есть обратное утверждение: произвольное решение $L(y) = 0$ может раскладываться на линейную комбинацию ФСР. *Доказывается (как по мне, с такой формулировкой и доказательства не надо) через построение системы уравнений с одинаковыми множителями линейной комбинации, где каждое уравнение содержит все части ФСР в виде производных от 0 в первом уравнении до $n-1$ в последнем, в правой части уравнений будет стоять наше произвольное решение (и также его производные). Определитель этой системы будет определителем Вронского, а значит, учитывая, что слева стоят элементы ФСР, определитель будет ненулевым, а значит сама система будет иметь единственное решение, причём это решение будет совпадать с тем, которое мы выбрали в начале для доказательства в силу единственности решения задачи Коши (это настолько страшно, что я всеми бы силами хотел избегать упоминания этого доказательства)*

**Т.** В однородном $L(y)$ не может быть больше $n$ линейно независимых решений. *Доказывается от противного: если $n$ решений уже линейно зависимы, то $n+1$ также будут зависимы. Если $n$ решений линейно независимы, значит это ФСР по определению, значит $n+1$-е решение будет выражаться из остальных по предыдущей теореме*

## Понижение порядка
Методы понижения степени:
1. Пусть в уравнении участвуют лишь производные от $m$ до $n$ и $m > 0$, то мы производим замену $u = y^{(m)}$. Решив диффур относительно $u$, производим обратную замену и интегрируем $m$ раз. (*очевидно, в решении будет $m$ произвольных постоянных, во всех последующих видах также будет больше одной константы*)
2. Уравнение, разрешённое относительно некоторой функции, явно не зависящей от $x$: $f(y, y', ..., y^{(n)}) = 0$. Здесь мы производим замену $y' = p(y) \rArr y'' = pp'$. **$p$ дифференцируется по $y$, которая является для $p$ аргументом.**. Найдя $p$, решаем диффур с разделяющимися переменными
3. Пусть уравнение однородно по группе всех производных, тогда проводим похожую замену на замену общей однородности: $y'(x) = y(x)z(x) \rArr y'' = zy' + yz' \rArr \frac{y''}{y} = z^2 + z'$
4. Выделение точной производной, *например*, мы замечаем $(y^3)' = 3y^2y'$

Методы 2 и 3 понижают порядок на единицу. Остальные могут и больше.

# 23.10.12 - лекция
## Неоднородные линейные диффуры с переменными коэффициентами
Очевидно, что вместо $L(y) = 0$ будет $L(y) = f(x)$. Как и для однородных, общее решение = частное решение + ФСР (*следует из линейности ФСР*)

Обратное утверждение также верно: если мы знаем решение неоднородного линейного диффура, то найдутся такие константы $C_1,...,C_n$, что они выделят из общего решения частное решение и ФСР. (*доказывается тривиально по предпоследней теореме с прошлой лекции*)

Если была найдена ФСР, то частное НЛДУПК (*ужасная аббревиатура, но я не хочу больше писать "неоднородное линейное дифференциальное уравнение с переменными коэффициентами"*) можно найти методом вариации постоянных:
- Представим частное решение через ФСР: $y_0(x) = C_1(x)y_1(x) + ... + C_n(x)y_n(x)$. Теперь наша цель - найти $C_i(x)$
- Дифференцируем частное решение: $y_0'(x) = C_1(x)y_1'(x) + ... + C_n(x)y_n'(x) + C_1'(x)y_1(x) + ... + C_n'(x)y_n(x)$ и потребуем, чтобы $C_1'(x)y_1(x) + ... + C_n'(x)y_n(x) = 0$ (*на третий раз перечитывания я наконец преисполнился в познании и понял, что это даже очень логично, ведь мы ищем в конечном счёте константы, хотя... Ай, неважно*). Вновь дифференцируем частное решение с тем же требованием. Продолжаем до производной порядка $n-1$
- Получаем следующие системы:
$$
\begin{cases}
  C_1'(x)y_1(x) + ... + C_n'(x)y_n(x) = 0 \newline
  C_1'(x)y_1'(x) + ... + C_n'(x)y_n'(x) = 0 \newline
  ... \newline
  C_1'(x)y_1^{(n-2)}(x) + ... + C_n'(x)y_n^{(n-2)}(x) = 0 \newline
\end{cases}
$$
$$
\begin{cases}
  y_0'(x) = C_1(x)y_1'(x) + ... + C_n(x)y_n'(x) \newline
  ... \newline
  y_0^{(n-1)}(x) = C_1(x)y_1^{(n-1)}(x) + ... + C_n(x)y_n^{(n-1)}(x)
\end{cases}
$$
- Из последней строчки второй системы дифференцированием получаем $n$-ю производную: $y_0^{(n)}(x) = C_1(x)y_1^{(n)}(x) + ... + C_n(x)y_n^{(n)}(x) + C_1'(x)y_1^{(n-1)}(x) + ... + C_n'(x)y_n^{(n-1)}(x)$ (**тут не обрезаем часть с производными от $C_i$-х!**)
- Подставляем вычисленные производные в исходное уравнение и, за счёт того, что $y_1,...,y_n$ - решения однородной версии, получаем в итоге просто $C_1'(x)y_1^{(n-1)}(x) + ... + C_n'(x)y_n^{(n-1)}(x) = f(x)$, которое добавляется в первую систему. Получаем:
$$
\begin{cases}
  C_1'(x)y_1(x) + ... + C_n'(x)y_n(x) = 0 \newline
  C_1'(x)y_1'(x) + ... + C_n'(x)y_n'(x) = 0 \newline
  ... \newline
  C_1'(x)y_1^{(n-2)}(x) + ... + C_n'(x)y_n^{(n-2)}(x) = 0 \newline
  C_1'(x)y_1^{(n-1)}(x) + ... + C_n'(x)y_n^{(n-1)}(x) = f(x) \newline
\end{cases}
$$
- Определитель этой системы будет определителем Вронского от линейно независимых решений уравнения $y_1,...,y_n$, а значит система имеет единственное решение, которое может быть найдено даже методом Крамера
- Решим систему, получаем $C_1'(x) = g_1(x), ..., C_n'(x) = g_n(x)$, интегрируем их и наконец **находим частное решение** $y_0(x)$

*Практическое применение рассмотрим на семинарах (надеюсь)*

Если правая часть НДУПК $f(x) = f_1(x) + ... + f_n(x)$, то частным решением будет сумма частных решений НДУПК с правой частью только из $f_i(x)$

## (Не)зависимость системы функций без определителя Вронского
**Определитель Грама** $\Gamma(y_1,...,y_m) = |A|$, где $a_{ij} = \int_a^b y_j(t)y_i(t)dt$.

Система непрерывных на $(a, b)$ функций линейно независима на $(a, b) \hArr \Gamma(\vec{y}) \ne 0$

**Л.** Если система функций линейно независима на интервале, то и любая её подсистема будет линейно независима. (*Доказательство: если предположить обратное, то найдутся такие $1 \le j \le k : a_j \ne 0 : a_1y_1 + ... + a_ky_k = 0$, но тогда при $k < i \le n : a_i = 0 : a_1y_1 + ... + a_ky_k + ... + a_ny_n = 0$, а это противоречит условию линейной независимости всей системы*)

**Л.** Если какая-либо из подсистем системы функций линейно зависима, то и вся система будет линейно зависима (*доказывается от противного банально по прошлой лемме*)

**Л.** Если функции линейно независимы на $(a,b) \rArr$ они линейно независимы на любом **объемлющем** интервале $(c,d)$, то есть таком, что $(a,b) \subseteq (c,d)$ (*Следует из факта, что линейно зависимые функции должны быть зависимы на всём интервале, а на его части $(a, b)$ они линейно независимы по условию*)

По линейно независимой системе непрерывно дифференцируемых $n$ раз функций $y_1, ..., y_n$ с определителем Вронского не равным нулю можно найти единственное однородное уравнение, решив систему вида:
$$
\begin{cases}
  y_1^{(n)} + p_1(x)y_1^{(n-1)} + ... + p_{n-1}(x)y_1' + p_n(x) y_1 = 0 \newline
  y_2^{(n)} + p_1(x)y_2^{(n-1)} + ... + p_{n-1}(x)y_2' + p_n(x) y_2 = 0 \newline
  ... \newline
  y_n^{(n)} + p_1(x)y_n^{(n-1)} + ... + p_{n-1}(x)y_n' + p_n(x) y_n = 0 \newline
\end{cases}
$$
*Решаться она будет без проблем методом Гаусса или Крамера (т.к. её определитель - это в точности определитель Вронского)*

**Л.** Если функции $y_1, ..., y_m$ - линейно независимые на $(a, b)$ решения НДУПК с $n \ge m$, то система решений будет также линейно независима на любом вложенном интервале/ (*Доказывается также от противного через тот факт, что если система зависима, то на любом её интервале найдутся ненулевые коэффициенты, при которых она будет равна нулю, а значит $y_0(x_0) = y_0'(x_0) = ... = y_0^{(n)} = 0$ - решение задачи Коши и при том единственное. Из этих утверждений приходим к выводу, что система от вложенного интервала никак не может быть зависимой при условии линейной независимости системы от бОльшего интервала*)

Один из таких методов - замена переменных, позволяющая перейти от уравнения с переменными коэффами к уравнению с постоянными.

Производим замену $x$ на $t = \xi(x)$ и $y(x)$ на $u(t) = y(\xi^{-1}(t))$. Далее пересчитываем производные и переходим к новому диффуру - уже с постоянными коэффициентами.

### Уравнение Эйлера
Общий вид:
$$
x^ny^{(n)} + a_1x^{n-1}y^{(n-1)} + ... + a_{n-1}xy' + a_ny = f(x)
$$
- $x = e^t, x > 0 \rArr t = \ln x$
- $x = -e^t, x < 0 \rArr t = \ln (-x)$

*Пример возьму с семинара*

Из достаточно страшных вычислений выводится общая формула для характеристического полинома уравнения Эйлера:
$$
p(\lambda) = \lambda(\lambda - 1)(\lambda - 2)...(\lambda - n + 1) + ... + a_{n-2}\lambda(\lambda - 1) + a_{n-1}\lambda + a_n
$$
**Найденные через него корни дадут нам ответы в виде суммы $Ce^{\lambda t}$, то есть далее мы должны произвести замену $t = \ln x$ и получить $Cx^{\lambda}$**

### Уравнение Лагранжа
Общий вид:
$$
(ax + b)^ny^{(n)} + a_1(ax + b)^{n-1}y^{(n-1)} + ... a_{n-1}(ax + b)y' + a_ny = f(x)
$$
*Замена тоже достаточно похожая:* $ax + b = e^t \rArr t = \ln |ax + b|$

Также частные решения уравнения Эйлера часто удобно искать после замены $y = x^{\lambda}$, где $\lambda$ - неизвестная постоянная

*Пример:*
$$
x^2y'' - 4xy' + 6y = 0 \newline
\rArr p(\lambda) = \lambda(\lambda - 1) - 4\lambda + 6 \newline
\lambda(\lambda - 1) - 4\lambda + 6 = 0 \newline
\lambda^2 - 5\lambda + 6 = 0 \newline
(\lambda - 2)(\lambda - 3) = 0 \newline
\rArr y = C_1e^{2t} + C_2e^{3t} \rArr y = C_1x^2 + C_2x^3 \newline
$$
*Альтернативный вариант:*
$$
x^2y'' - 4xy' + 6y = 0 \newline
y = x^{\lambda} \rArr y' = \lambda x^{\lambda - 1} \rArr y'' = \lambda(\lambda - 1) x^{\lambda - 2} \newline
x^2\lambda(\lambda - 1) x^{\lambda - 2} - 4x\lambda x^{\lambda - 1} + 6x^{\lambda} = 0 \newline
x^{\lambda}[x^2\lambda(\lambda - 1) x^{-2} - 4x\lambda x^{-1} + 6] = 0 \newline
x^{\lambda}[\lambda(\lambda - 1) - 4\lambda + 6] = 0 \newline
x^{\lambda} > 0, \space\space\space
\lambda(\lambda - 1) - 4\lambda + 6 = 0 \newline
... аналогично ...
$$

### Уравнение Чебышева

## Система линейных диффуров первого порядка
Общий вид:
$$
\begin{cases}
  y_1' = a_{11}(x)y_1 + ... + a_{1n}(x)y_n + f_1(x) \\
  ... \\
  y_n' = a_{n1}(x)y_1 + ... + a_{nn}(x)y_n + f_n(x) \\
\end{cases}
$$
- $a_{ij}(x)$ - коэффициенты
- $f_i(x)$ образуют в совокупности правую часть системы
- И те, и другие, предполагаются непрерывными на $(a, b)$, где рассматривают диффур
- Матрица коэффициентов $A(x)$ называется **матрицей системы**
- $Y(x)$ - вектор-столбец неизвестных
- $F(x)$ - вектор-столбец правой части
- С этими обозначениями можем записать систему в виде: $\frac{dY}{dx} = A(x)Y + F(x)$
- $L[Y] = \frac{dY}{dx} - A(x)Y$ - линейный дифференциальный оператор над множеством вектор-функций высоты $n$

Многие утверждения для ЛДУ порядка $n$ справедливы и для системы ЛДУ первого порядка, в частности:
- Задача Коши имеет единственное решение
- $\forall i \in [1, n] : f_i(x) \equiv 0 \rArr$ система **однородная**

Линейная зависимость для системы вектор-функций $Y_1, ..., Y_n$ определяется также, как и для функций

Определитель матрицы, склеенной из столбцов $Y_1, ..., Y_n$ будет **определителем Вронского для системы вектор-функций**

Теоремы об определителе Вронского также аналогичны вплоть до доказательств:
- Для зависимой системы вектор-функций определитель Вронского будет тождественно равен нулю
- Если вектор-функции $Y_1, ..., Y_n$ независимы на интервале $(a, b)$ и являются решением однородной системы $\frac{dY}{dx} = A(x)Y$, тогда их определитель Вронского не равен нулю ни в одной точке
- Если определитель Вронского для системы решений вектор-функций однородной системы равен нулю в одной точке, то он равен нулю во всех точках интервала
- Если определитель Вронского для системы решений вектор-функций однородной системы не равен нулю в одной точке, то он не равен нулю во всех точках интервала
- Для линейной независимости системы вектор-функций решений однородной системы диффуров необходимо и достаточно, чтобы её определитель Вронского не равнялся нулю хотя бы в одной точке интервала

**Т.** Формула Остроградского-Лиувилля системы решений вектор-функций однородной системы диффуров:
$$
W(x) = W(x_0)e^{\int_{x_0}^x [\sum_{i=1}^n a_{ii}(ы)]dы}
$$
*Доказывается через дифференцирование определителя, в результате которого приходим к выражению $W'(x) = [\sum_{i=1}^n a_{ii}(x)]W(x)$, которое после интегрирования и даст нам формулу (хз как, если честно, но да...)*

Определение ФСР аналогично: система из $n$ линейно независимых на интервале $(a, b)$ решений однородной системы диффуров

**Т.** Для любой однородной системы найдётся ФСР. *Доказывается похоже на тоже утверждение для ЛДУ порядка $n$, но теперь мы берём для всех вектор-функций решений символ Кронекера (фактически, также получаем диагональную матрицу)*

Последующие теоремы также аналогичны вплоть до доказательств:
- Линейная комбинация ФСР также даст нам решение однородной системы
  - Обратное: любое решение системы ЛДУ можно представить как линейную комбинацию ФСР
- Однородная система из $n$ диффуров имеет ровно $n$ линейно независимых решений (*тут даже Васкевич решил уже не приводить доказательство, хех*)

*А дальше чуть ли не десять страниц повторяются те же определения, но уже в виде векторов, а не матриц-столбцов (что ведь вообще считай что тоже самое!). Ну ладно... Кое-что выпишу*

**Фундаментальная матрица решений** - матрица, в которой столбцами являются вектор-функции ФСР

### Решение через с.з., с.в.
*А теперь повторяем про с.з. и с.в. (хм... А ведь в том году мы вроде и правда проходили это только на семинарах)*

$P_n(l) = |A - lE|$ - **характеристический полином** матрицы $A$

**Собственные значения** - корни характеристического полинома

**Собственные вектора** - нетривиальные (ненулевые) вектор-решения $\vec{x}_k$ уравнений $(A - l_kE)\vec{x} = 0$. Соответственное, с.в. относятся к определённым с.з.

Для поиска с.в. следует брать алгебраические дополнения какой-либо строки матрицы $A - l_kE$, причём выбирать стоит максимально простые, не забывая о необходимости нетривиальности

Если все с.в. матрицы $A$ линейно независимы, говорят, что матрица имеет **простую структуру**

Для однородной системы $Y' = AY$ вектор-функции ФСР: $\vec{y_i}(t) = \vec{v}_ie^{l_it}$, где $\vec{v}_i$ - с.в. матрицы $A$, $l_i$ - с.з матрицы $A$

Соответственно, общим решением будет линейная комбинация всех вектор-функций ФСР с произвольными коэффициентами

При линейной независимости никаких дополнительных действий не требуется

Если с.з. будет комплексным, то и соответствующий с.в. будет комплексным, а также будет сопряжённый ему. Учитывая суммирование векторов на финальном этапе решения, имеет смысл взять два вещественных корня: вещественную и мнимую части решения $\vec{v}_ie^{l_it}$ (доп. пояснения см. в [семинаре](#231030---семинар))

В случае, если есть кратные с.з., но с.в. разные, решения определяются просто, будто бы с.з. разные (*в соответствии с формулой на 3 абзаца выше*)
- Способ проверки: $n - m = r$, где $r$ - кратность с.з. $l_k$, $m$ - ранг матрицы $A - l_kE$

Если же с.в. меньше, чем ранг матрицы ($n - m < r$), то ищем нужное количество присоединённых векторов $v_i$-х (см. [семинар](#231031---семинар)) и записываем следующие элементы ФСР: 
$$
\vec{u}_ke^{l_kt} \\
(\vec{v}_{k1} + t\vec{u}_k)e^{l_kt} \\
(\vec{v}_{k2} + t\vec{v}_{k1} + \frac{t^2}{2!}\vec{u}_k)e^{l_kt} \\
... \\
(\vec{v}_{kr} + t\vec{v}_{k(r-1)} + \frac{t^{r-1}}{(r-1)!}\vec{u}_k)e^{l_kt}
$$

## Линейные уравнения с переменными коэффициентами
### Уравнение Эйлера
Уравнение вида $\sum_{i=0}^n a_i t^i y^{(i)} = 0$

Проводим замену $t = e^s, t > 0$ и $t = -e^s, t < 0$

$$
u(s) = y(\ln s) = y(t) \newline
\frac{dy}{dt} = \frac{du}{ds}\frac{ds}{dt} = u'\frac{1}{t} \newline
\frac{d^2y}{dt^2} = \frac{d}{dt}(u'\frac{1}{t}) = \frac{-1}{t^2}u' + (u''\frac{1}{t})\frac{1}{t} = \frac{1}{t^2}(u'' - u') \newline

## Устойчивость ОДУ и их систем
Теория устойчивости исследует поведение решений диффуров и систем при больших значениях времени $t$ (*здесь независимая переменная обозначается так*)

Здесь рассматривается вектор-функция $f(t, y) = (f_1(t, y_1, ..., y_n), ..., f_n(t, y_1, ..., y_n))$

Задача Коши формулируется в таком случае к поиску решения для системы $y' = f(t, y)$ в начальный момент времени $y(t_0) = y_0$

Решение $y = \phi(t)$ задачи Коши **устойчиво**, если $\forall \epsilon > 0 : \exist \delta > 0 : \forall y(t)$ - другого (*возможно?*) решения системы, удовлетворяющего условию $|y(t_0) - \phi(t_0)| \le \delta$ одновременно для всех $t \ge t_0$ выполняется $|y(t) - \phi(t)| \le \epsilon$. Если при этом выполняется предельное равенство $\lim_{t \rarr \infty} |y(t) - \phi(t)| = 0$, то решение $\phi(t)$ **асимптотически устойчиво**

Решение задачи Коши **неустойчиво**, если для этого решения $\phi(t) : \exist \epsilon_0 > 0 : \forall \delta > 0$ (сколь угодно малого) и $\forall y(t) \ne \phi(t)$ - другого решения задачи Коши с условием $|y(t_0) - \phi(t_0)| \le \delta$, найдётся такой момент времени $t_1 = t(\delta) \ge t_0 : |y(t_1) - \phi(t_1)| \ge \epsilon_1$

**Задача об устойчивости** сводится к выяснению, является ли решение данной задачи Коши устойчивым, асимптотически устойчивым или неустойчивым, при этом задачу всегда можно свести к проверке на устойчивость тождественно нулевого решения:
- Исходные условия: $y' = f(t, y), y(t_0) = y_0$. $\phi(t)$ - решение
- Замена $z(t) = y(t) - \phi(t)$
- Получаем: $z' = f(t, z + \phi) - f(t, \phi), z(t_0) = 0$
- Полученное таким образом тождественно нулевое решение обладает тем же видом устойчивости, что и исходное $\phi(t)$

Геометрически решение задачи Коши представляет собой функцию, описывающую **интегральную кривую** в пространстве $\R^{n+1}$ с координатами $(t, y_1, ..., y_n)$, проходящую через заданную точку. В механике решение будет описывать перемещение материальной точки с течением времени $t$ в пространстве $\R^n$ с координатами $(y_1, ..., y_n)$. Это пространство называется **фазовым**, а кривые - **фазовыми траекториями**, которые представляют собой проекции интегральных кривых. Вектор-функцию $f(t, y)$ будут тогда называть **фазовой скоростью**

Нулевое решение задачи Коши будет проецироваться в фазовое пространство в точку, совпадающую с началом координат.

Геометрически нулевая интегральная кривая будет **устойчивой**, если $\forall \epsilon > 0 : \exist \delta > 0 : \forall y(t)$ - любая траектория, начинающаяся в момент времени $t_0$ в шаре $|y(t_0)| < \delta$ с течением времени не выходит из шара $|y(t)| < \epsilon$. Если траектория при этом стремится к началу координат, то можно говорить об **асимптотической устойчивости**

**Т.** Если все с.з. матрицы $A$ в системе диффуров $Y' = AY$ имеют отрицательные вещественные части, то нулевое решение системы асимптотически устойчиво, если же хотя бы одно с.з. имеет положительную вещественную часть, то нулевое решение неустойчиво

*Доказывается* анализом примера с системой из двух диффуров и случаев:
- Когда оба с.з. - вещественные числа и различны
- Когда с.з. комплексные (а значит сопряжённые)
- Когда с.з. вещественные и равны

*Выписав формулы для решений этих случаев, можно заметить, что благодаря частям $e^{lt}$ при отрицательных $l$ решение действительно будет асимптотически устойчивым, при хотя бы одном положительном оно может быть любого типа в зависимости от произвольных констант, при нулевом значении $l$ решение часто будет устойчивым, но уже не асимптотически

Если решение диффура $y' = f(t, y)$ проецируется в точку фазового пространства, то это решение называется **точкой покоя** или её **положением равновесия**. Вектор $y_0 = (y_1^0, ..., y_n^0)$ называется точкой покоя системы $y' = f(t, y)$ тогда и только тогда, когда $y_0$ является стационарным решением системы $f_1(t, y_1^0, ..., y_n^0) = 0, ..., f_n(t, y_1^0, ..., y_n^0) = 0$

Для систем диффуров выделяют такую классификацию точек покоя:
- **Устойчивый узел** - все траектории системы начинаются в некоторой фиксированной окрестности точки покоя и неограниченно к ней приближаются вместе с касательными
  - **Устойчивый фокус** - касательные не входят в точку покоя
- **Неустойчивый узел** - все траектории начинаются в сколь угодно малой окрестности точки покоя и устремляются в бесконечность вместе с касательными
  - **Неустойчивый фокус** - касательные не уходят в бесконечность
- **Седло** - среди траекторий, начинающихся в сколь угодно малой окрестности точки покоя, есть как стремящиеся в точку, так и стремящиеся к бесконечности
- **Центр** - все начинающиеся в $\delta$-окрестности точки покоя кривые замкнуты и не покидают заданную $\epsilon$-окрестность ($\epsilon > 0, \delta = \delta(\epsilon)$)

### Теоремы Ляпунова
Пусть функции $f_1(t, y), ..., f_n(t, y)$ устойчивы и непрерывны в полуцилинде:
$$
\{(t, y) | t_0 \le t < +\infty, ||y|| \le \epsilon_0\} \\
$$
$$
\epsilon_0 > 0, y = (y_1, ..., y_n), ||y|| = \sqrt{y_1^2 + ... + y_n^2}
$$
А также при $||y|| \le \epsilon_0$ определена некоторая непрерывно дифференцируемая функция $V(y) = V(y_1, ..., y_n)$

Учитывая, что $y' = f(t, y)$, получаем:
$$
V'(y) = \sum_{i=1}^n f_i(t, y) \frac{\delta V(y)}{\delta y_i}
$$

Пусть при $||y|| \le \epsilon_0$ определена некоторая непрерывно дифференцируемая функция $V(y)$. $V(y) > 0$ при $||y|| > 0$, $V(0) = 0$ и всюду в области определения в силу системы диффуров производная $V'(y)$, тогда $V(y)$ - **функция Ляпунова**

## Устойчивость
Для системы уравнений $y' = f(t,y)$ и задачи Коши для $t_0, y_0$. Будем говорить, что решение задачи - $\tilde{y_0}(t)$ **устойчиво (по Ляпунову)**, если:
1. $\tilde{y}(t)$ определено на $(t_0, \infty)$
2. Для той же системы будет задача Коши $(t_0, y_1)$ с решением $\tilde{y_1}(t)$:
   - $\exist \Delta > 0 : ||y_1 - y_0|| < \Delta \rArr \tilde{y_1}(t)$ определено на $(t_0, \infty)$
3. $\forall \epsilon_p > 0 : \exist \delta = \delta(\epsilon_p) : ||y_1 - y_0|| < \delta \rArr \forall t > t_0 : ||\tilde{y_0}(t) - \tilde{y_1}(t)|| < \epsilon_p$

*Человеческая интерпретация:* если для системы диффуров выполняются условия 1-3, то решение задачи Коши будет устойчивым, то есть при малой погрешности входных данных будет малая погрешность результатов.

**Асимптотическая устойчивость** для решения $\tilde{y_0}(t)$ системы уравнений $y' = f(t,y)$ и задачи Коши для $t_0, y_0$ будет, если:
1. $\tilde{y_0}(t)$ устойчиво по Ляпунову
2. $\exist \delta > 0 : ||y_1 - y_0|| \le \delta \rArr ||\tilde{y_0}(t) - \tilde{y_1}(t)|| \rarr 0$

**Спектральный критерий Ляпунова об устойчивости** - нулевое решение системы будет асимптотически устойчивым, если спектр содержится строго в левой полуплоскости и Жорданова клетка, отвечающая только мнимым корням, имеет размер 1. Критерий не плох лиш для очень малых матриц, для прикладных задач необходимо использовать матричные уравнения Ляпунова

**Комлпексное число** - упорядоченная пара вещественный, чисел $(a, b)$, где $a$ - **вещественная** часть $Re z$, $b$ - **мнимая** $Im z$: $z = a + ib$. Числа только с вещественной частью **отождествляются с вещественной прямой**, число только с мнимой частью называется **мнимым**. $i$ - **мнимая единица**

Вещественные числа равны $\hArr$ равны их вещественные и мнимые части соответственно

Сами комплексные числа обозначаются $\Complex$, с определёнными для них операциями сложения и умножения $\Complex$ становится полем.

Комплексные числа могут быть представлены точками в $\R^2$, которое называется **комплексной плоскостью** и может быть также обозначено $\Complex$. Если убрать из этой плоскости нуль, получим **проколотую в нуле комплексную плоскость**

Для $z = x + iy$, **сопряжённым** будет $\overline{z} = x - iy$

$z\overline{z}$ всегда положительно. $|z| = \sqrt{z\overline{z}}$ называется **модулем комплексного числа**

Если $z \ne 0$, то определён угол $\phi$ между радиус-вектором этого комплексного числа и осью $OX$, который называется **аргументом комплексного числа**. Определяется с периодом $2\pi k$. Значение аргумента, удовлетворяющее $-\pi < \arg z < \pi$ называется **главным**

Для комплексных чисел не определено отношение порядка, но их можно **сравнивать по модулю**. Определяется однозначно

Таким образом, мы можем обозначить комплексное число, через аргумент и модуль. Это будут **полярные координаты**. Благодаря им, достаточно удобно обозначать разность комплексных чисел векторным методом

**Неравенства треугольника:**
- $|z_1 + z_2| \le |z_1| + |z_2|$
- $|z_1 - z_2| \ge ||z_1| - |z_2||$
- Двойное: $||z_1| - |z_2|| \le |z_1 \plusmn z_2| \le |z_1| + |z_2|$

**Окрестностью комплексного числа** $z_0$ будем обозначать круг радиуса $\delta > 0$ такой, что $C(\delta, z_0) = O_{\delta}(z_0) = \{ z \in \Complex : |z - z_0| < \delta \}$. Есть также проколотая версия - будут входить все точки, кроме $z_0$ - в обозначении добавляется звёздочка

**Окрестность бесконечно удалённой комплексной точки** - внешность круга положительного радиуса $R$: $O_{R}(\infty) = \{ z \in \Complex : |z| > R \}$

Определения **изолированной**, **предельной**, **граничной**, **внутренней** и **внешней** точки даются аналогично таковым для вещественных (*ещё бы помнить их...*)

Совокупность граничных точек множества $G$ обозначается $\delta G$ и называется **границей**.

Множество $G \subset \Complex$ **ограничено**, если все его точки лежат в некотором круге конечного радиуса

Множество $G \subset \Complex$ **замкнуто**, если в объединении со своей границей равняется самому себе. При этом $\overline{G} = G \cup \delta G$ называется замыканием

Множество $G \subset \Complex$ называется **открытым**, если для любого его числа существует дэльта-окрестность, принадлежащая множеству

**Связанное множество** - множество, в котором можно между двумя любыми точками провести кривую. **Выпуклое множество** - между любыми двумя точками можно провести прямую. (*ясное дело, кривая и прямая полностью лежат в множестве*)

**Область** - открытое связное множество

**Замкнутая область** - объединение области и её границы

**Расширенная комплексная плоскость** - к ней добавляется бесконечность. Будет одновременно замкнутым и открытым (*как?..*)

Граница любого множества - замкнутое множество

**Принцип Больцано-Вейрштрасса** - любое бесконечное множество точек расширенной комплексной плоскости имеет хотя бы одну предельную точку.

**Лемма Гейне-Бореля** - из бесконечного открытого покрытия ограниченного замкнутого множества комплексной плоскости можно всегда можно выделить конечное покрытие

**Лемма о вложенных прямоугольниках** - пусть есть последовательность $\{P_n\}$ замкнутых прямоугольников со сторонами, параллельными осям координат, причём $P_{n+1} \subset P_n$, а также последовательность длин диагоналей прямоугольников стремится к нулю, тогда существует единственная точка $z_0$, принадлежащая пересечению всех этих прямоугольников

**Предел для комплексных чисел** (*как я понял, по Вейрштрассу*):
- последовательность $\{z_n\}$ называется **сходящейся к $z_0$**, если $\forall \epsilon > 0 : \exist N = N(\epsilon) : \forall n \ge N : |z_n - z_0| < \epsilon$
- При этом пишут $\lim_{n \rarr \infty} z_n = z_0$
- $\lim_{n \rarr \infty} z_n = 0$ и $\lim_{n \rarr \infty} |z_n| = 0$ **эквивалентны**
- Геометрически можно сказать, что все точки с некоторого номера принадлежат эпсилон-окрестности $z_0$-го

**Лемма**: $\lim_{n \rarr \infty} x_n + iy_n = x_0 + iy_0 \hArr \lim_{n \rarr \infty} x_n = x_0 \And \lim_{n \rarr \infty} y_n = y_0$

При помощи этой леммы все теоремы о пределах для $\R$ становятся справедливы и для $\Complex$

**Критерий Коши**: $\forall \epsilon > 0 : \exist N = N(\epsilon) : \forall n \ge N : \forall m \in \N : |z_{n+m} - z_n| < \epsilon$

Последовательность сходится к бесконечности, если $\forall R > 0 : \exist N = N(R) : \forall n \ge N : |z_n| > R$
- $\lim_{n \rarr \infty} z_n = \infty \hArr \lim_{n \rarr \infty} |z_n| = +\infty$
- Если $z_n \ne 0 \rArr \lim_{n \rarr \infty} z_n = \infty \hArr \lim_{n \rarr \infty} 1/z_n = 0$

Ряд комплексных чисел $\sum_{k=1}^{\infty} z_k$ сходится, если сходится последовательность его частичных сумм. Причём $\lim_{k \rarr \infty} S_k = S$ - **сумма ряда**
- **Необходимое условие сходимости ряда**: $\lim_{k \rarr \infty} z_k = 0$
- **Критерий Коши сходимости ряда**: $\forall \epsilon > 0 : \exist N = N(\epsilon) : \forall n \ge N : \forall m \in \N : |\sum_{k=1}^{m} z_{n+k}| < \epsilon$ (*фактически, модуль разности частичных сумм раскрыли вот в такую запись*)

Ряд сходится **абсолютно**, если сходится и ряд его модулей. Если сходится сам ряд, но не ряд его модулей, тогда ряд сходится **условно**
- Абсолютная сходимость ряда эквивалентна сходимости рядов его вещественных и мнимых частей
- Члены абсолютно сходящего ряда можно менять местами, не меняя суммы ряда
- Если ряды $\sum_{k=1}^{\infty} z_k = S_1$ и $\sum_{k=1}^{\infty} w_k = S_2$ сходятся абсолютно, то ряд $\sum_{k=1}^{\infty} (z_1w_k + z_2w_{k-1} + ... + z_kw_1)$ сходится абсолютно, причём его сумма равна $S_1S_2$

Определение ряда позволяет ввести основные функции комплексной переменной:
- $e^z \equiv \sum_{k=0}^{\infty} \frac{z^k}{k!} = 1 + z + \frac{z^2}{2!} + ...$
- $\cos z \equiv \sum_{k=0}^{\infty} (-1)^k\frac{z^2k}{(2k)!} = 1 - \frac{z^2}{2!} + \frac{z^4}{4!} + ...$
- $\sin z \equiv \sum_{k=0}^{\infty} (-1)^k\frac{z^{2k+1}}{(2k+1)!} = z - \frac{z^3}{3!} + \frac{z^5}{5!} + ...$

По признаку Даламбера (*деление следующего члена ряда на предыдущий*) все эти ряды сходятся при любых $z$

$\rArr$ **показательная форма**: $z = |z|e^{i\arg z}, z \ne 0$
- А также: $e^z = e^{x + iy} = e^xe^{iy} = e^x(\cos y + i\sin y) \rArr |e^z| = e^x \ne 0, |z| < +\infty$

## Функциональные ряды
Для представления приближённого значения $f(z)$ используются функциональные и степенные ряды.

**Функциональный ряд** на множестве $D$ комплексной плоскости называется ряд вида $\sum_{k=1}^{\infty} f_k(z)$. Функциональный ряд сходится в $D$, если он сходится в любой точке $D$ $\rArr l = \lim_{n \rarr \infty} S_n(z) : S_n(z) = \sum_{k=1}^n f_k(z)$

Функциональный ряд **сходится равномерно** в области $D$, когда $\forall \epsilon > 0 : \exist N = N(\epsilon) : \forall n > N : \forall z \in D : |S_N(z) - S_n(z)| < \epsilon$ (мы говорим о том, что он сходится именно **равномерно**, так как $N$ у нас общее для любых $z$)

**Т. "Признак равномерной сходимости"** Пусть каждая функция $f_i(z)$ функционального ряда, начиная с некоторого $i_0$ удовлетворяет оценкам $\sup_{z \in D}|f_i(z)| \le \alpha_i$ и при этом числовой ряд $\alpha_i$-х сходится равномерно и абсолютно. *За счёт сходимости ряда альф и неравенства треугольника + определения сходимости ряда может записать следующую оценку $|\sum_{k=1}^m f_{n+k}(z)| \le \sum_{k=1}^m |f_{n+k}(z)| \le \sum_{k=1}^m \alpha_{n+k} < \epsilon$. Пользуясь ей и критерием Коши заключаем, что ряд сходится равномерно и абсолютно*

**Т.** Сумма $S(z)$ равномерно сходящегося на ограниченном замкнутом множестве $D$ функционального ряда, в котором все функции $f_i(z)$ непрерывны на $D$, тогда $S(z)$ также будет непрерывной на $D$ функцией. *Доказательство:* возьмём точку $z_0 \in D$ и докажем непрерывность $S(z) - S(z_0) = S(z) - S_n(z) + S_n(z) - S_n(z_0) + S_n(z_0) - S(z_0) \rArr |S(z) - S(z_0)| = |S(z) - S_n(z)| + |S_n(z) - S_n(z_0)| + |S_n(z_0) - S(z_0)|$. $\exist N_0 = N_0(S) : \forall n > N_0 \rArr \sup_{z \in D}|S(z) - S_n(z)| < \epsilon / 3$ - для доказательства этой оценки, которая подтвердит равномерную сходимость возьмём оценку $n = N_0 + 1$ и из свойства равномерности сходящего ряда мы подтверждаем истинность выражения $\sup_{z \in D}|S(z) - S_n(z)| < \epsilon / 3$, а из него следует $S(z) - S(z_0) < \epsilon$

## Cтепенные ряды
Общий вид степенного ряда для комплексных чисел $\sum_{k=0}^{\infty} C_kz^k$. При $z = 0$ ряд будет равняться $C_0$.

**Т. Коши-Адамара** - $l = \overline{\lim}_{k \rarr \infty} |C_k|^{1/k}$:
- $l = 0 \rArr$ сопоставленный этим $C_i$-м степенной комплексный ряд сходится абсолютно во всей комплексной плоскости, а на любом компакте - сходиться равномерно
- $l = \infty \rArr$ ряд расходится в любой точке $z$, не равной началу координат
- $0 < l < \infty \rArr$ ряд сходится абсолютно, но лишь при условии $|z| < 1 / l$, расходится при $|z| > 1 / l$ и ведёт себя неопределённо в точках на границе этой окружности

*Доказательство:*
- Для первого случая мы говорим, что общий предел последовательности $|C_i|$-х будет существовать и $\ge 0$ и $\le l \rArr \lim_{k \rarr \infty} |C_k|^{1/k} = 0 \rArr \lim_{k \rarr \infty} |C_kz^k|^{1/k} = |z| \lim_{k \rarr \infty} |C_k|^{1/k} = 0$, а далее по признаку сходимости ряда Коши получаем, что и исходный степенной ряд будет сходиться
- *Остальные случаи доказываются аналогично*

**С. "Первая теорема Абеля"**. Если степенной ряд сходится в $z_0 \ne 0$, то он сходится абсолютно $\forall z : |z| < |z_0|$. *Доказательство:* мы получаем по условию Т. Коши-Адамара $l \ne 0$, при этом ряд абсолютно сходится, значит $|z_0| \le 1/l$, а значит и $|z| < 1/l \rArr$ ряд будет абсолютно сходиться

$r = 1/l$ - **радиус сходимости ряда**. Если вместо $l$ записать предел, то получим равенство Коши-Адамара

## Производная комплексной функции
Комплексная функция $f(z) = w$ **дифференцируема в некоторой точке из области $D$**, если $\exist$ конечный $\lim_{\Delta z \rarr 0} \frac{f(z + \Delta z) - f(z)}{\Delta z} = \lim_{\Delta z \rarr 0} \frac{\Delta w}{\Delta z}$. Такой предел будет называться **производной комплексной функции** и будет обозначаться штрихом.

Ввиду двумерности комплексных чисел нам безразлично, с какой стороны мы стремимся к $z$

Представим $\Delta z = \Delta x + i\Delta y$, тогда можем определить такие производные:
- $\Delta y = 0 \rArr f'(z) = \lim_{\Delta x \rarr 0} [\frac{u(x + \Delta x, y) - u(x, y)}{\Delta x} + i\frac{v(x + \Delta x, y) - v(x, y)}{\Delta x}] = \frac{\delta u}{\delta x} + i\frac{\delta v}{\delta x}$
- Аналогичным образом (*почти*) определяется производная при $\Delta x = 0, \Delta y \rarr 0$: $f'(z) = \lim_{\Delta x \rarr 0} [\frac{u(x, y + \Delta y) - u(x, y)}{i\Delta y} + \frac{v(x, y + \Delta y) - v(x, y)}{\Delta y}] = \frac{\delta u}{i\delta y} + \frac{\delta v}{\delta y} = -i\frac{\delta u}{\delta y} + \frac{\delta v}{\delta y}$

Из этих двух формул мы можем вывести **условия Коши-Римана**:
- $\frac{\delta u}{\delta x} = \frac{\delta v}{\delta y}$
- $\frac{\delta v}{\delta x} = -\frac{\delta u}{\delta y}$

*Для примера можно легко проверить дифференцируемость функции $f(z) = z$*

**Для дифференцируемой функции условия Коши-Римана будут выполняться, но не для любой удовлетворяющей этим условиям функции будет существовать производная, то есть условия необходимые, но не достаточные** (*пример - функция $f(z) = e^{-\frac{1}{z^4}}, z \ne 0$ из-за разрыва в нуле*)

**Л.** Для функции $f = u + iv$, если $u(x,y)$ и $v(x,y)$ дифференцируемы в точке $z = x + iy$ и при этом выполняются условия Коши-Римана, то функция $f$ дифференцируема

*Доказательство:*

Функции $u$, $v$ всюду дифференцируемы, а значит:
- $\Delta u = \frac{\delta u}{\delta x}\Delta x + \frac{\delta u}{\delta y}\Delta y + o(||\Delta z|)$
- $\Delta v = \frac{\delta v}{\delta x}\Delta x + \frac{\delta v}{\delta y}\Delta y + o(||\Delta z|)$

**Формальные производные** - особые обозначения (**не путать с полной производной**)
- $\frac{\delta}{\delta z} = \frac{1}{2}(\frac{\delta}{\delta x} - i\frac{\delta}{\delta y})$
- $\frac{\delta}{\delta \overline{z}} = \frac{1}{2}(\frac{\delta}{\delta x} + i\frac{\delta}{\delta y})$

Отсюда заключаем, что:
- $\Delta x = \frac{1}{2}(\Delta z + \Delta \overline{z})$
- $\Delta y = \frac{1}{2i}(\Delta z - \Delta \overline{z})$

Представим $\Delta f$ в виде: $\Delta f = \Delta u + i \Delta v = (\frac{\delta u}{\delta x} + i\frac{\delta v}{\delta x})\Delta x + (\frac{\delta u}{\delta y} + i\frac{\delta v}{\delta y})\Delta y + o(\Delta z) = \frac{\delta f}{\delta z}\Delta z + \frac{\delta f}{\delta \overline{z}}\Delta \overline{z} + o(\Delta z)$ `(*)`

Домножая формальные производные на $f = u + iv$, получаем:
- $\frac{\delta f}{\delta z} = \frac{1}{2}[\frac{\delta u}{\delta x} + \frac{\delta v}{\delta y} + i(\frac{\delta v}{\delta x} - \frac{\delta u}{\delta y})]$

\begin{cases}
  f(z), z \in D \\
  0, z \in \Complex - \overline{D} 
\end{cases}
$$
*Доказательство:*

Для начала, при $z \in \Complex - \overline{D}$, $\frac{f(w)dw}{w - z}$ будет аналитична, а значит по теореме Коши, равна нулю

При $z \in D$. Определим круг радиуса $\delta > 0$, лежащий в $D$ вокруг точки $z$. Кривая круга будет описываться формулой $|w - z| = \delta$, тогда, по обобщённой теореме Коши получаем $\int_G \frac{f(w)dw}{w - z} = \int_{|w - z| = \delta} \frac{f(w)dw}{w - z}$

Преобразуем $\int_{|w - z| = \delta} \frac{f(w)dw}{w - z} = \int_{|w - z| = \delta} \frac{f(w) - f(z)}{w - z}dw + f(z) \int_{|w - z| = \delta} \frac{dw}{w - z}$

Первый интеграл после равенства при такой замене в пределе $\delta \rarr 0$ даёт 0

Таким образом, при переходе к пределу получаем $\int_{|w - z| = \delta} \frac{f(w)dw}{w - z} = 2i\pi f(z) \rArr \int_G \frac{f(w)dw}{w - z} = 2i\pi f(z)$

### Применение интеграла типа Коши
Если функция аналитическая в области $D$, значит она будет бесконечно в ней дифференцируема. Докатывается подбором такого круга, который будет лежать в $D$, а далее, в силу интегральной формулы Коши: $f(z) = \frac{1}{2i\pi} \int_{|w - z_0| < \delta} \frac{f(w)dw}{w - z}$ - интеграл типа Коши, а значит, определяемая им функция будет бесконечно дифференцируема

**Т. Морера** Пусть $f(z)$ - непрерывная в области $D$ функция, если для любой замкнутой кусочно гладкой Жордановой кривой $\Gamma \subset D$ справедливо равенство $\int_{\Gamma} f(z)dz = 0$, то $f(z)$ аналитическая в области $D$ функция. *Доказывается похоже на прошлое утверждение:* берём точки в $D$: $z_0, z$, соединим их кривой $\Sigma$, то $\int_{\Sigma} f(w)dw$ не зависит от направления интегрирования, а зависит только от $z, z_0$. Значит $\int_{\Sigma} f(w)dw = \int_{z_0}^z f(w)dw = F(z)$:
$$
\rArr F(z + \Delta z) - F(z) = \int_{z_0}^{z + \Delta z} f(w)dw - \int_{z_0}^z f(w)dw = \int_z^{z+\Delta z} f(w)dw
$$
Через оценку сверху для выбранного в области $D$ круга убеждаемся, что $F$ дифференцируема, а значит дифференцируема и $f$

Множество аналитических функций таких, что $\Phi'(z) = f(z)$, где $f(z)$ - также аналитичная в $D$, то $\Phi(z)$ - **неопределённый интеграл $f(z)$**. Существенно условие аналитичности, так как просто для непрерывных функций в комплексных числах интеграла не будет

Неопределённый интеграл и интеграл типа Коши будут отличаться друг от друга на константу

Все первообразные также будут отличаться на константу (**комплексную**)

Формула Неютона-Лейбница для комплексных чисел также работает аналогично

## Ряды Тейлора
**Т. Тейлора**. Аналитичная в $D$ функция $f(z)$ в окрестности каждой точки $z_0 \in D$ представима в виде суммы степенного ряда $f(z) = \sum_{k=0}^{\infty} c_k(z - z_0)^k$, причём радиус сходимости этого ряда **не меньше** расстояния от $z_0$ до границы $G = \delta D$

*Доказательство:*
Возьмём $0 < \delta < \rho = dist(z_0, G)$, где $G = \delta D$, таким образом, в круге $|z - z_0| < \delta$ к $f(z)$ применима интегральная формула Коши, в которой представляем:
$$
\frac{1}{w - z} = \frac{1}{(w - z_0)(1 - \frac{z - z_0}{w - z_0})}
$$
И замечаем, что
$$
\frac{1}{1 - \frac{z - z_0}{w - z_0}} = \sum_{k=0}^{\infty} (\frac{z - z_0}{w - z_0})^k
$$
Тогда
$$
\frac{1}{w - z} = \frac{1}{w - z_0}\sum_{k=0}^{\infty} (\frac{z - z_0}{w - z_0})^k
$$
$$
\rArr f(z) = \frac{1}{2i\pi} \int_{|w - z_0| = \delta} \frac{f(w)dw}{w - z} = \\
= \frac{1}{2i\pi} \int_{|w - z_0| = \delta} \frac{f(w)dw}{w - z_0}\sum_{k=0}^{\infty} (\frac{z - z_0}{w - z_0})^k = \\
= \frac{1}{2i\pi} \sum_{k=0}^{\infty} [\int_{|w - z_0| = \delta} \frac{f(w)dw}{(w - z_0)^{k+1}} (z - z_0)^k] = \sum_{k=0}^{\infty} c_k(z - z_0)^k \\
c_k = \frac{1}{2i\pi} \int_{|w - z_0| = \delta} \frac{f(w)dw}{(w - z_0)^{k+1}} = \frac{f^{(k)}(z_0)}{k!}
$$
Отсюда получаем:
$$
f(z) = \sum_{k=0}^{\infty} \frac{f^{(k)}(z_0)}{k!}(z - z_0)^k
$$
Этот ряд будет сходиться в круге $|z - z_0| < \rho$ (*сказано это доказать... Я бы предложил воспользоваться формулой Стерлинга и признаком Коши-Адамара*)

Такой ряд будет называться **рядом Тейлора**

*Как это доказывает теорему? Ну... $|z - z_0| < \rho$ означает, что не меньше, единственность следует из опредления $c_k$-х*

Если функция аналитична, то коэффициенты её ряда Тэйлора могут быть определны через интегральную формулу Коши (*смотри выше*), при этом имеет место оценка: $|c_k| \le \frac{1}{R^k}\max_{|w - z| = R} |f(w)|$ - **неравенство Коши**

## Неопределённый инграл функции комплксной переменной
Неопределённый интеграл от $f$ - это такая $Ф(z)$, что $Ф'(z) = f$

$$
\sum_{k=0}^{\infty} c_k \frac{1}{(z - z_0)^k} \\

$$

*Дописать с фотки формулы*

**Т. Лиувилля** Если ограниченная функция всюду аналитична, тогда $f = const$ (*доказывается через неравенство Коши и теорему Тейлора*)

$$
f(z) = \sum_{k=-\infty}^{-1} c_k(z - z_0)^k + \sum_{k=0}^{\infty} c_k(z - z_0)^k
$$
- Правый ряд - правильная часть - Тейлор
- Левый ряд - главная  часть - Лорановский

$$
f(z) = \sum_{k=-\infty}^0 c_k z^k + \sum_1^{\infty} c_kz^k
$$
Части Тейлора и Лоарна меняются местами. *Фактически, мы тут производим замену $1/z_0$*

## Ряд Лорана
Рассматриваем **функциональный ряд по отрицательным степеням** $\sum_{k=-1}^{-\infty} b_k (z - z_0)^k$. Каждый член ряда имеет смысл при всех $z \ne z_0$

Заменой $w = \frac{1}{z - z_0}$ ряд приводится к степенному $\sum_{k=1}^{\infty} b_{-k} w^k$. Пусть его радиус сходимости равен $r > 0$, а $S_1(w)$ - его сумма в круге $|w| < r$, тогда ряд будет сходиться абсолютно в $|z - z_0| = \frac{1}{|w|} > 1/r > 0$ и его сумма $S_2(z) = S_1(\frac{1}{z - z_0})$

Пусть функциональный ряд $\sum_{k=0}^{\infty} c_k (z - z_0)^k$ сходится в круге $|z - z_0| < R$ и имеет сумму $S_1(z)$, тогда функция $S(z) = S_1(z) + S_2(z) = \sum_{k=0}^{\infty} c_k (z - z_0)^k + \sum_{k=-1}^{-\infty} b_k (z - z_0)^k = \sum_{k=-\infty}^{\infty} c_k (z - z_0)^k$, где $c_{-k} = b_k$ является аналитической в кольце $r < |z - z_0| < R$

**Теорема Лорана**. Аналитическая в кольце $r < |z - z_0| < R$ функция $f(z)$ представима в каждой точке этого кольца в виде **ряда Лорана** 
$$
\sum_{k=-\infty}^{\infty} c_k (z - z_0)^k \\
c_k = \frac{1}{2i\pi}\int_{|w - z_0| = \delta} \frac{f(w)dw}{(w - z_0)^{k+1}}
$$ 
*Вот почему нельзя было и тут записать через производные и факториал?..*

*Доказательство:*
Возьмём кольцо $r_1 < |z - z_0| < R_1$ внутри основного кольца и представим по интегральной формуле Коши
$$
f(z) = \frac{1}{2i\pi} \int_{|z - z_0| = R_1} \frac{f(w)dw}{w - z} - \frac{1}{2i\pi} \int_{|z - z_0| = r_1} \frac{f(w)dw}{w - z}
$$
В первом интеграле представим $\frac{1}{w - z}$ (*таким же образом, как в теореме Тейлора*) как 
$$
\frac{1}{w - z_0}\sum_{k=0}^{\infty} (\frac{z - z_0}{w - z_0})^k
$$
Во втором интеграле представим $\frac{1}{w - z}$ как 
$$
- \frac{1}{w - z_0}\sum_{k=0}^{\infty} (\frac{z - z_0}{w - z_0})^k
$$
Подставляя эти представление в формулу Коши и вынося суммирование наружу **получаем искомую формулу**

Ряд с нуля до бесконечности называется **правильной** частью, с $-1$ до $-\infty$ - **главной** (иногда её могут писать как $\sum_{k=1}^{\infty} \frac{c_k}{(z - z_0)^k}$)

Если функция $f(z)$ аналитична в окрестности бесконечной точки ($|z| > R$), то функция $g(w) = f(\frac{1}{w})$ будет аналитична при $0 < |w| < \frac{1}{R}$, а значит будет раскладываться в ряд Лорана:
$$
g(w) = \sum_{k=0}^{\infty} b_kw^k + \sum_{k=-1}^{-\infty} b_k \frac{1}{w^k}
$$
Полагая $w = 1/z$, получаем разложение $f(z)$ в ряд Лорана в оркестности бесконечно удалённой точки (*и, по идее, тут у нас местами меняются главная и правильная части, но точной уверенности в этом моменте у меня до сих пор нет...*)

В фиксированном кольце аналитическая в нём функция раскладывается в ряд Лорана единственным образом

*Далее со стр. 7 14-й лекции идёт пара очень полезных примеров*

Ряд Лорана связан с рядом Фурье. Рассмотрим кольцо $-\epsilon < |z| < 1 + \epsilon, \epsilon > 0$, тогда функция раскладывается в ряд Лорана в точке $z_0 = 0$, причём $c_k = \frac{1}{2i\pi}\int_{|z| = 1}\frac{f(w)dw}{w^{n+1}}$, поэтому можем представить $w = e^{i\phi} \rArr c_n = \frac{1}{2\pi} \int_0^{2\pi} f(e^{i\phi})e^{-ik\phi} d\phi$

За счёт этой замены представим ряд Лорана в ином виде:
$$
\sum_{k=-\infty}^{\infty} c_k e^{ik\phi} = c_0 + \sum_{k=1}^{\infty} (c_ke^{ik\phi} + c_{-k}e^{-ik\phi})
$$
Далее проводим замены:
- $a_0 = 2c_0$
- $a_k = c_k + c_{-k}$
- $b_k = i(c_k - c_{-k})$
- $h(\phi) = f(e^{i\phi})$

Из чего получаем:
$$
h(\phi) = \frac{a_0}{2} + \sum_{k=1}^{\infty} (a_k \cos \phi + b_k \sin \phi) \\
a_0 = \frac{1}{\pi}\int_0^{2\pi} h(\phi)d\phi \\
a_k = \frac{1}{2\pi}\int_0^{2\pi} h(\phi)\cos \phi d\phi \\
b_k = \frac{1}{2\pi}\int_0^{2\pi} h(\phi)\sin \phi d\phi
$$

Таким образом, на единичной окружности ряд Лорана для $f(z)$ является рядом Фурье для суперпозиции $h(\phi) = f(e^{i\phi})$

## Изучение особых точек функции
Нужно для счёта интегралов

$z$ - собоая точка, если $f(z)$ не определена. Пример: $1/z$ при $z = 0$

**Изолированная особая точка** - в какой-то окрестности нет других особых точек. Отсюда следует и определение **неизолированной** - невозможно подобрать окрестность, в которой нет других особых точек.

Пример: $\frac{1}{\sin z}$, здесь $\pi k$ - изолированные особые точки, $\infty$ - неизолированная

Работаем только с **изолированными особыми точками**, которые делятся на:
1. Устранимые: $\exist \lim_{z \rarr z_0} f(z) < \infty$ - главная часть ряда Лорана отсутствует
2. Полюс порядка $k$: $\lim_{z \rarr z_0} f(z) = \infty$ либо в главной части $k$ слагаемых (можно представить в виде $\frac{1}{(z - z_0)^k}\phi(z)$, где $\phi(z)$ - аналитичная)
3. Существенно-особая точка: либо предела не существует, либо главная часть имеет бесконечное число слагаемых

**Критерий полюса**:
1. $z_0$ - полюс порядка $k$, если $\lim_{z \rarr z_0}(z - z_0)^{k-1}f(z) = \infty$, но $\lim_{z \rarr z_0}(z - z_0)^kf(z) = w_0$ - конечное комплексное число
2. Аналитическая функция $f(z)$, $z_0$ - нуль порядка $k$, если $f(z_0) = f'(z_0) = ... = f^{{k-1}}(z_0) = 0$ и $f^{(k)} \ne 0$
   1. Для функции $f(z) = \frac{\phi(z)}{\psi(z)}$, если $z_0$ для $\phi(z)$ - нуля порядка $m$, а для $\psi(z)$ - порядка $n$, то при $m \ge n$ ИОТ будет **устранимой**, если же $m < n$, то ИОТ - **полюс порядка $n - m$** (архиполезный метод на практике, потому что позволяет без проблем вычислить штуки типа $\frac{z}{\sin z}$)

При разложении на ряд Лорана в бублике главная часть будет с отрицательными степенями, а при разложении во внейшней части (бублик до бесконечности), тогда в главной части будут положительные степени

## Вычет
**Вычет** функции $f(z)$ в точке $z_0$:
$$
Res_{z = z_0} f(z) = \frac{1}{2i\pi}\int_{\gamma} f(z)dz 
$$
$\gamma$ - область с одной особой точкой $z_0$

$$
Res_{z = \infty} \frac{1}{z} = -1
$$
**ВАЖНЫЙ ПРИМЕР**

**Формулы вычетов:**
1. Для конечной устранимой точки выечт будет 0
2. Для полюса порядка $k$: $Res_{z = z_0}f = \frac{1}{(k-1)!}\lim_{z\rarr z_0}\frac{d^{k-1}}{dz^{k-1}}((z-z_0)^kf(z))$. В частности, для просто полюса получаем $Res_{z = z_0}f = \lim_{z\rarr z_0}(z-z_0)f(z)$
3. Для просто полюса ($k = 1$): работаем с аналитичным числителем $\phi$ и знаменателем $\xi$, производная котрого не равна нулю: $Res_{z = z_0}f = \frac{\phi(z_0)}{\xi'(z_0)}$
4. Для существенной ОТ вычет будет равен $C_{-1}$ (коэффициент при первом члене главной части ряда Лорана, то есть том члене, где у нас $z^{-1}$) (**на самом деле это применимо для любого вычета, но в силу особенности ряда Лорана для других ОТ применяется нечасто**)
5. Для бесконечной ОТ **ЛЮБОГО ТИПА** вычет равен $-C_{-1}$

**Основная теорема о вычетах**. $f(z)$ аналитична в $D$ за исключением **конечного** числа изолированных особых точек, при этом в $D$ определена кусочно гладка кривая $G$ и $f(z)$ непрерывна в $G \cup D$ тогда:
$$
\int_D f(z)dz = 2i\pi \sum_{i \in I} Res_{z = z_i} f
$$
$I$ - индексы особых точек

## Преобразования Лапласа
Функция называется **оригиналом**, если существует такое вещественное $a$, что
$$
\int_0^{+\infty} |f(t)|e^{-at}dt < +\infty
$$
**Порядком роста** оригинала называется точная нижняя граница множества всех таких вещественных чисел $a$. Обозначается как $a(f)$

Для заданного оригинала $f(t)$ можно определить функцию $F(z), z = c + id$
$$
F(z) = \int_0^{+\infty} f(t)e^{-zt}dt
$$
Этот интеграл будет сходиться при условии, что $Re z = c > a(f)$. Таким образом, область определения будет представлять собой полуплоскость

Функцию $F(z)$ будут называть **изображением** оригинала $f(t)$

**Преобразование Лапласа** сопоставляет оригиналу его изображение. $F(z) = L(f(t))$

- Функция Хевисайда: 1 при $t \ge 0$, иначе - 0. Изображение: $1/z$
- $L(\cos t) = \frac{z}{1 + z^2}$
- $L(\sin t) = \frac{1}{1 + z^2}$

Не всякая гладкая функция будет оригиналом. Для таких преобразование Лапласа не определено. Например, $e^{t^2}$ или $1/t$. Удобно предполагать, что их порядок роста $+\infty$

При этом функция $\frac{1}{\sqrt{t}}$ стремится к бесконечности при $t \rarr +0$, но имеет преобразование Лапласа, а её порядок равняется нулю

Для функции $e^{-t^2}$ порядок роста будет $-\infty$

**Л. Критерий оригинала**. Если $f(t)$ определена и непрерывна при $t \ge 0$ и при этом существуют такие постоянные $M > 0$ и $k \ge 0$, что $\forall t \ge 0 : |f(t)| \le Me^{kt}$

**С.** Оригиналами будут функции вида $e^{wt}, w \in \Complex$ и $t^n$, из преобразования Лапласа вычисляются по формулам $L(e^{wt}) = \frac{1}{z - w}$ и $L(t^n) = \frac{n!}{z^{n+1}}$

**Т. Об аналитичности изображения**. Для функции с порядком роста $a(f)$ её изображение будет абсолютно сходиться при $Re z > a(f)$, при этом, если $Re z \rarr +\infty$, то $F(z) \rarr 0$

Если же $F(z)$ аналитична в окрестности бесконечной точки, то $F(z) \rarr 0$ при $z \rarr \infty$, то есть в любом направлении

Свойства преобразования Лапласа:
- Линейность (следует сразу из линейности интегрирования) (при этом для двух преобразований общая сходимость будет определяться более высоким порядком роста)
- **Теорема подобия**: если $F(z) = L(f(t)), q > 0 \rArr L(f(qt)) = \frac{1}{q}F(\frac{z}{q})$
- **Теорема смещения**: если $F(z) = L(f(t)), w \in \Complex \rArr L(e^{-wt}f(t)) = F(z + w)$
- **Теорема запаздывания**: если $F(z) = L(f(t)), a > 0$, тогда функция
$$
f_a(t) = \begin{cases}
  0, t < a \\
  f(t - a), t \ge a
\end{cases}
$$
также является оригиналом, причём $L(f_a(t)) = e^{-za}F(z)$. Так, при помощи двух функций Хевисайда мы можем определить функцию единичного импульса $\phi(t) = H(t) - H_a(t)$, причём $L(\phi(t)) = \frac{1 - e^{-za}}{z}$

Пусть функция $u(t)$ непрерывно дифференцируема, причём $u, u'$ - оригиналы с одинаковыми параметрами критерия оригинала, а также $\forall T > 0 : \forall t_1, t_2 \in [0, 2T] : |u'(t_2) - u'(t_1)| \le N(T)\sqrt{|t_2 - t_1|}$

Обозначим $u_o(z) = L(u(t))$, тогда для всех $t \in [t_0, T], t_0 > 0$ имеет место равенство:
$$
|u(t) - \frac{1}{2i\pi}\int_{a - ib}^{a + ib} u_o(z)e^{zt}dz| \le \frac{M_1}{b}
$$
где $b > 0, a > k$ ($k$ - показтель из критерия оригинала), $M_1$ не зависит от $b$

При $b \rarr +\infty$ получаем формулу обращения преобразования Лапласа:
$$
u(t) = \frac{1}{2i\pi}\int_{a - i\infty}^{a + i\infty} u_o(z)e^{zt}dz
$$

**Т. о производной оригинала**. Если функция и её производная непрерывны при $t \ge 0$ и являются оригиналами, то из простого интегрирования по частям следует формула:
$$
L(f(t)) = \frac{f(0)}{z} + \frac{L(f'(t))}{z}
\rArr L(f'(t)) = zL(f(t)) - f(0)
$$
**С.** Если $f(t)$ дифференцируется $n$ раз и при этом $f^{(n)}(t)$ - оригинал, то и $f(t)$ - оригинал, причём
$$
L(f^{(n)}(t)) = z^nL(f(t)) - z^{n-1}f(0) - z^{n-2}f'(0) - ... - f^{(n-1)}(0)
$$
