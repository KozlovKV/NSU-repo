# 23.09.07 - лекция
## Основные понятия
Мы будем говорить только об обыкновенны дифференциальных уравнения (*БОЛЬШОЕ СПАСИБО*)

**О.** Обыкновенное дифф. уравнение порядка $n$ относительно функции $y(x)$ - уравнение вида $F(x, y, y', ..., y^{(n)}) = 0$. В дифф.уре всегда есть производные. Порядком уравнения называется порядок старшей производной. Разделение диффуров на классы производится по их порядку. Для разных классов есть разные методы решения.

**О.** Решение диффура - это функция $y = \phi(x)$, если при подстановке в уравнение $y \rArr \phi, y' \rArr \phi'$ получается тождество. Как правило уравнение имеет бесконечное количество решений, которые, также, зависят от произвольных констант C, такое решение записывается как $\phi(x, C_1, C_2, ..., C_n)$ - такая запись называется **общим решением диффура**

Часто в ходе преобразований мы получаем лишь **неявное задание** решения диффура вида $\Phi(x, y, C_1, C_2, ..., C_n) = 0$. Обычно перейти к явному решению от неявного мы можем лишь локально

**Нормальная форма** - форма диффура, в которой мы старшую производную выражаем через остальные производные ($y^{(n)} = f(x, y, y', ..., y^{(n-1)}$)

**О.** График решения диффура $y = \phi(x)$ называется **интегральной кривой**

**Поле направлений** для системы $y' = f(x, y)$ - множество всех прямых, проходящих через точки $(x, y)$ ООФ под углом, тангенс которого будет равняться $f(x, y)$. Для интегральной кривой диффура поле её касательных совпадает с полем направлений этого уравнения.

*Не уверен, что важно хоть где-то, но пусть будет:* для уравнений второго порядка всякая интегральная кривая определяется связью между координатами точек этой кривой, наклоном касательной и кривизной в каждой её точке

**Для системы диффуров большинство определений аналогичны**

**Задача Коши для заданного ОДУ** состоит в том, чтобы из всех решений найти такое решение $y = y(x)$, которое вместе со своими производными $n - 1$ порядка принимает заданные значения $y_0, y_1, ..., y_n$ при заданном значении независимой переменной $x_0$

**Траектория ОДУ первого порядка** - $(\phi_1(t), \phi_2(t), ..., \phi_n(t))$ - при изменении параметра $t$ эта точка будет описывать кривую $n$-мерном пространстве. Каждая координата точки задаётся решением ОДУ: $\forall i \le n: \phi_{i1}(x) = y_{i1}(x), \phi_{i2}(x) = y_{i2}(x), ..., \phi_{im}(x) = y_{im}(x)$. Соответственно, для систему ОДУ задача Коши сводится к поиску подходящей траектории

**Не у всякой задачи Коши есть решение и не у всякой оно единственное.** Решение, в котором в каждой точки нарушается условие единственности, называется **особым**

ОДУ порядка $n$ может быть преобразовано в систему ОДУ первого порядка путём не очень хитрого переобозначения: $y \harr y_1, y' \harr y_2, ..., y^{(n-1)} \harr y_n \rArr y_{(n-1)}' = y_n$. Считая, что исходное ОДУ записано в нормальной форме, получаем из него систему:
$$
\begin{cases}
  y_1' = y_2 \newline
  y_2' = y_3 \newline
  ... \newline
  y_{(n-1)}' = y_n \newline
  y_n' = f(x, y_1, y_2, ..., y_n)
\end{cases}
$$

Возможен и обратный процесс, то есть переход от систему ОДУ высокого порядка к одному ОДУ более низкого порядка. Для этого требуется существование первых частных производных для функции одного из уравнений системы, а также возможность выразить одну незивестную функцию через другую

*Скипнул до простейших классов диффуров и методов их решения*

## Простейшие классы диффуров и методы их решения
**Уравнения с разделяющимися переменными** - уравнения вида $y' = f(x)g(y)$ либо $f_1(x)g_1(y)y' = f_2(x)g_2(y)$ - в них нет подфункций, зависящих сразу от обеих переменных. Путём преобразований (деления) и понимания, что $y' = \frac{dy}{dx}$ переходим к форме, в которой слева и справа остаются только переменные одного имени, затем интегрируем обе половины и получаем решение - **PROFIT** (**Однако необходимо учитывать, что во время преобразований мы можем потерять режи... *Стоп, что?..* Решение мы можем потерять, поэтому то, на что мы делили, надо будет отдельно подставить для проверки**)

Если переменные находятся под одной и той же функцией и не могут разделены, то мы производим замену выражения под функцией. Пример: уравнение $y' = f(ax + by + c)$. Берём $z = ax + by + c$, считаем $dz = adx + bdy \rArr dy = \frac{dz - adx}{b}$, подставляем в исходное уравнение:
$$
\frac{dz - adx}{bdx} = f(z) \hArr \frac{dz}{bdx} - \frac{a}{b} = f(z) \hArr z' = a + bf(z)
$$
Далее решаем диффур по обычным правилам.

## Уравнения с однородной правой частью (однородное уравнение)
Выглядит:
$$
y' = h(\frac{y}{x})
$$
$h()$ - однородная функция степени $k$, если $\forall C > 0 : h(Cx) = C^k h(x)$. Если однородна по группе переменных $y_1, ..., y_n$, то $\forall C > 0 : h(x, ..., Cy_1, ..., Cy_n) = C^kh(x, ..., y_1, ..., y_n)$

Для проверки однородности можно дописать произвольный коэффициент ко всем $x, y$. Если все коэффициенты могут быть сокращены, значит уравнение однородное.

# 23.09.14 - лекция
## Ещё один вид однородных диффуров
К однородным диффурам можно привести следующие диффуры:
$$
y' = f(\frac{a_1x + b_1y + c_1}{a_2x + b_2y + c_2})
$$

В случае, если прямые $a_1x + b_1y + c_1 = 0$ и $a_2x + b_2y + c_2 = 0$ пресекаются в точке $(x_0, y_0)$, делают замену $x = x_0 + t, y = y_0 + z(t)$. Уравнение в таком случае преобразуется к виду:
$$
y' = f(\frac{a_1t + b_1z}{a_2t + b_2z})
$$
Здесь $f$ - однородная функция степени 0

Если прямые $a_1x + b_1y + c_1 = 0$ и $a_2x + b_2y + c_2 = 0$ не пересекаются (то есть параллельны), то переходим путём алгебраических преобразований к уравнению вида $y' = \tilde{f}(a_1x + b_1y)$, которое приводится к разделяющемуся заменой $z = a_1x + b_1y$

*Пример:*
$$
y' = \frac{x - y + 1}{x + y - 3} \newline
\begin{cases}
  x - y + 1 = 0 \newline
  x + y - 3 = 0
\end{cases} \newline
x_0 = 1, y_0 = 2 \rArr x = t + 1, y = z + 2 \newline
z' = \frac{t - z}{t + z}
$$
Получили однородное уравнение, которое решается классической заменой вида $z = tu$

**Иногда уравнение приводится к однородному заменой $y = z^m(x)$**, при этом $m$ - неизвестное число, с которым полученное уравнение должно удовлетворять условию однородности. $m$ ищется отдельно (*Пример немного непонятен на этапе поиска $m$. Надеюсь, на семинаре мы рассмотрим этот тип*)

## Линейные дифференциальные уравнения
**О.** Диффуры первого порядка вида $y' + p(x)y = q(x)$ называются **линейными диффурами первого порядка**. $p(x)$ - **коэффициент диффура**, $q(x)$ - его правая часть

Метод решения (**метод вариации постоянной** или **метод Лагранжа**):
1. Решаем уравнение, где рассматриваем $q(x) = 0$. В таком случае у нас будет простое однородное уравнение. Берём полученный ответ вида $y = C\phi(x)$
2. Для $q(x) \not= 0$
   1. Делаем подстановку решения однородного уравнения при $q(x) = 0$, заменяя константу на неизвестную $y = C(x)\phi(x)$
   2. Поиск решения будет сводиться к поиску $C(x)$ классическими методами

Некоторые диффуры можно привести к линейному виду, представив $x(y)$, то есть поменять местами зависимую и независимую переменные (*ясное дело, мы не просто меняем букву производной, а проводим преобразования, держа в голове, что $y'^{-1} = x'$*) $(2e^y - x)y' = 1 | / y' \rArr 2e^y - x = x' \rArr x' + x = 2e^y$

Уравнение вида $f'(y)y' + a(x)f(y) = b(x)$ также приводится к линейному заменой $z = f(y) \rArr z' + a(x)z = b(x)$

*Пример:*
$$
\frac{y}{\sqrt{y^2 + 1}}y' + \sqrt{y^2 + 1} = x^2 + 1 \newline
z = \sqrt{y^2 + 1} \rArr z' = \frac{yy'}{\sqrt{y^2 + 1}} \newline
z' + z = x^2 + 1
$$

Уравнение вида $y' + p(x) = q(x)e^{ny}, n \not = 0$ сводится к линейному заменой $z(x) = e^{-ny(x)} \rArr z'(x) = -ne^{-ny(x)}y'(x) = -nz(x)y'(x)$. Тривиальными преобразования получаем $\frac{z'}{-n} + p(x)z = q(x)$

### Уравнение Бернулли
**Уравнение Бернулли** - диффур вида $y' + p(x)y = q(x)y^a$ при $a \not \in {0, 1}$ (*при этих значения мы получаем простой линейный диффур*). Для перехода к линейному диффуру производим замену $z(x) = y^{1 - a}(x) \rArr z'(x) = (1 - a)y^{1 - a - 1}(x)y'(x)$: $z' + (1 - a)p(x)z = (1 - a)q(x)$.

Также при $a > 0$ **дополнительным решением уравнения Бернулли** будет $y = 0$

### Уравнение Рикатти
**Уравнение Рикатти** - диффур вида $y' + a(x)y + b(x)y^2 = c(x)$. В общем случае не решается в элементарных функциях, но если мы знаем частное решение $y = y_1(x)$, то можем провести замену $y = y_1 + z$, которая сведёт уравнение Рикатти к уравнению Бернулли, которое сведётся к линейному заменой $u = \frac{1}{z}$

## Замена на $z^m$ для однородности
В лекции было про такие. Подбираем $m$ для $y = z^m$ так, чтобы $y' = mz^{m-1}z' = ...$ степень всех мономов совпадало.

*Пример:*
$$
2x^2y' = y^3 + xy \newline
y = z^m \rArr y' = mz^{m-1}z' \newline
2x^2mz^{m-1}z' = z^{3m} + xz^m \newline
2+m-1 = 3m = m + 1 \rArr 2m = 1 \rArr m = \frac{1}{2} \rArr y = z^{\frac{1}{2}}, y > 0 \newline
\frac{x^2z'}{z^{\frac{1}{2}}} = z^{\frac{3}{2}} + xz^{\frac{1}{2}} \newline
$$
Получили классическое однородное уравнение:
$$
x^2z' = z^2 + xz |/x^2 \not = 0\newline
z' = \frac{z^2 + xz}{x^2} \newline
z = ux \rArr z' = u'x + u \newline
u'x + u = \frac{u^2x^2 + ux^2}{x^2} \newline
$$

## Уравнение в полных дифференциалах
**Уравнение в полных дифференциалах** - уравнение вида $M(x, y)dx + N(x, y)dy = 0$, то есть его левая часть (*очевидно, речь о преобразовании, где мы получаем слева только $\frac{dy}{dx}$*) является полным дифференциалом некоторой непрерывно дифференцируемой $F(x, y)$

Для таких уравнений имеет место равенство: 
$$
dF = \frac{\delta F}{\delta x}dx + \frac{\delta F}{\delta y}dy
$$
А также должно выполняться равенство:
$$
dF = M(x, y)dx + N(x, y)dy
$$
Отсюда следует, что:
$$
\frac{\delta F}{\delta x} = M(x, y), \space\space\space \frac{\delta F}{\delta y} = N(x, y)
$$
Для всех этих равенств необходимое и достаточное условие:
$$
\frac{\delta M(x, y)}{\delta y} = \frac{\delta N(x, y)}{\delta x}
$$
Если верно последнее, то решаться диффур будет через последовательное интегрирование:
$$
\frac{\delta F}{\delta x}(x, y) = M(x, y)
$$
Интегрируем это равенство по переменной $x$, считая $y$ константой:
$$
F(x, y) = \int M(x, y) + \phi(y) = Ф(x, y) + \phi(y)
$$
$\phi(y)$ - произвольное слагаемое, зависящее от $y$, по ней и будет дифференцировать в обратную сторону
$$
\phi'(y) = (F(x, y) - Ф(x, y))' = N(x, y) - \frac{\delta Ф}{\delta y}
$$
**Правая часть здесь обязательно будет зависеть только от $y$**. Интегрируем это уравнение и находим $\phi(y)$ и находим подстановкой в $F(x, y) = Ф(x, y) + \phi(y)$

*Человеческая интерпретация:*
1. У нас есть утверждения о частных производных вида $\frac{\delta F}{\delta x} = M(x, y)$ и $\frac{\delta F}{\delta y} = N(x, y)$
2. Проводим интегрирование одной из них по той переменной, от которой взять частный дифференциал, другую переменную обозначаем как зависимую "константу"
3. Вычислив таким образом частичную форму $F(x, y)$ с зависимой "константой", подставляем её слева во второй частную производную из пункта 1 и считаем его
4. В итоге нам остаётся решить тривиальное дифференциальное уравнение, в котором будет только $y$

**Не всякое уравнение вида $M(x, y)dx + N(x, y)dy = 0$ является уравнением в полных дифференциалах**, то есть не все удовлетворяют условию $\frac{\delta M(x, y)}{\delta y} = \frac{\delta N(x, y)}{\delta x}$, однако почти всегда существует такая функция $\mu(x,y)$, что при умножении на неё $M,N$ уравнение придёт к этому виду, то есть выполнится условие:
$$
\frac{\delta}{\delta y}(\mu(x, y)M(x, y)) = \frac{\delta}{\delta x}(\mu(x, y)N(x, y))
$$
В таком случае $\mu(x,y)$ - **интегрирующий множитель**. Если он известен и не обращается в нуль, то диффур решается по изложенной выше схеме

Для поиска $\mu$ надо вычислить уравнение:
$$
\frac{\delta (\mu M)}{\delta y} = \frac{\delta (\mu N)}{\delta x} \newline
\frac{\delta \mu}{\delta y}M + \frac{\delta M}{\delta y}\mu = \frac{\delta \mu}{\delta x}N + \frac{\delta N}{\delta x}\mu
$$
Делим на $\mu$ и, где она останется, вносим её под дифференциал:
$$
\frac{\delta \ln\mu}{\delta y}M + \frac{\delta M}{\delta y} = \frac{\delta \ln\mu}{\delta x}N + \frac{\delta N}{\delta x}
$$
Подставляем значения $M, N, \frac{\delta M}{\delta y}$ и $\frac{\delta N}{\delta x}$, а далее находим решения, зависящие только от одной переменная (*убираем слагаемое с частным дифференциалом по другой*) и таким образом находим $\frac{\delta \ln\mu}{\delta y}$ либо $\frac{\delta \ln\mu}{\delta x}$, а далее вычисляем значение $\mu$, интегрируя по нужной переменной

## Уравнения первого порядка, неразрешимые относительно производной
Общий вид уравнений первого порядка неразрешённых относительно производной таков: $F(x, y, y') = 0$. 

В простейшем случае левая часть уравнение элементарно представляется в разрешённом относительно производной $F(x, y, y') = (y' - f_1(x,y))(y' - f_2(x,y))...(y' - f_m(x,y))$, при этом диффур распадается на $m$ диффуров вида $y' = f_1(x,y), ..., y' = f_m(x,y)$ - решение каждого из них будет решением исходного диффура, то есть будет ветвью общего решения. Кроме того, решениями будут частные особые решения этих систем, а также решения, полученные склеиванием нескольких ветвей решения.

## Параметрическое решение дифференциальных уравнений
Диффур $F(x, y, y') = 0$ Допускает запись в параметрической форме, если $\exist \phi(u,v), \psi(u,v), \chi(u,v)$ параметров $u,v$ такие, что имеет место тождество $F(\phi(u,v), \psi(u,v), \chi(u,v)) \equiv 0$

Из записи, справедливой для любого решения диффура - $dy = y'dx$ - получаем через подстановки следующую:
$$
\frac{d\psi}{du}du + \frac{d\psi}{dv}dv = \chi(u, v)(\frac{d\phi}{du}du + \frac{d\phi}{dv}dv)
$$
Относительно переменных $(u,v)$ мы получаем обыкновенный диффур в симметричном виде.

Если решение было найдено в форме $v = w(u, C)$, то мы получаем:
- $x = \phi(u, w(u, C))$
- $y = \psi(u, w(u, C))$

*Очевидным образом эти 2 записи преобразуются, если у нас было найдено $u = w(v, C)$*

*Практическое применение метода затрудняется поиском этих самых параметрических функций*

Однако есть 2 случая, в которых поиск параметрических функций сильно упрощён:
- Если $F(x, y, y') = 0$ разрешим относительно $y$: $y = \psi(x, y') \rArr \phi = u = x, \chi = v = y'$
- Если $F(x, y, y') = 0$ разрешим относительно $x$: $x = \phi(y, y') \rArr \psi = u = y, \chi = v = y'$

## Решением диффуров высокого порядка
[Про понижение смотри в семинаре](#231010---семинар)

*Сначала идут записи для простейших диффуров порядка $n$, которые необходимо последовательно интегрировать $n$ раз, чтобы получить ответ вида $y = a_1x^{n-1} + a_2x^{n-2} + ... + a_{n-1}x + a_n$*

### Линейный диффур высокого порядка
$$
y^{(n)} + a_1(x)y^{(n-1)} + a_2(x)y^{(n-2)} + ... + a_{n-1}(x)y' + a_n(x)y = f(x)
$$
**Линейное уравнение порядка $n$**. Также может записываться в **операторной форме**
$$
\frac{d^n y}{dx^n} + a_1(x)\frac{d^{n-1} y}{dx^{n-1}} + ... a_{n-1}(x)\frac{dy}{dx} + a_n(x)y = f(x)
$$
Если в этой форме $a_i$ - константы, то говорят **линейном уравнении с постоянными коэффициентами**

*Также существует ещё одна максимально извращённая запись с подстановкой $D = \frac{d}{dx}$:*
$$
(D^n + a_1D^{n-1} + .... + a_{n-1}D + a_n)y = f(x)
$$

**Характеристический полином** $p(\lambda) = \lambda^n + a_1 \lambda^{n-1} + ... + a_{n-1}\lambda + a_n$ будет характеристическим полиномом уравнения, если $a_i$ - постоянные уравнения, равны $a_j$ полинома.

Если полином $p(\lambda)$ имеет $k$ вещественных и комплексных корней, то, оказывается соответствующий ему диффур в операторной форме (при $f(x) = 0$, то есть для **однородных уравнений**) можно записать в **факторизованной форме**
$$
(D - \lambda_1)(D - \lambda_2)...(D - \lambda_k)y = 0
$$
*Доказывается через пример для порядка `2`*

### Формула общего решения для линейных уравнений с постоянными коэффициентами
Для однородного линейного диффура в факторизованной форме порядка $n$ решениями будут $y_1 = C_1e^{\lambda_1x}, ..., y_n = C_ne^{\lambda_n x}$, а также любая их линейная комбинация. Если все $\lambda$ различны, то **линейная комбинация всех $y_i$** будет описывать **общее решение диффура**.

*Пример*:
$$
\frac{d^2 y}{dx^2} - \frac{dy}{dx} - 2y = 0 \newline
p(\lambda) = \lambda^2 - \lambda - 2 
\rArr \lambda_1 = -1, \lambda_2 = 2 \newline
(D + 1)(D - 2)y = 0 \rArr y = C_1e^{-x} + C_2e^{2x}
$$

Если у нас совпадают некоторые корни характеристического полинома $\lambda_1 = \lambda_2 = ... = \lambda_k = \lambda$, тогда вместо сумму в общем решении их слагаемые преобразуются в запись вида $(C_1 + C_2x + ... + C_kx^{k-1})e^{\lambda x}$

Для комплексно сопряжённых корней $\lambda_1 = a + bi, \lambda_2 = a - bi$ решением примет вид $e^{ax}[C_1\cos bx + C_2\sin bx]$, если таковых сопряжённых пар $k$ штук, получим $e^{ax}[(C_1 + C_2x + ... C_kx^{k-1})\cos bx + (B_0 + B_1x + ... + B_kx^{k-1})\sin bx]$

## Линейного неоднородного диффура высокого порядка
Общая форма диффура была описана выше. Алгоритм такой:
- Находим решение однородного диффура
- Находим любое частное решение неоднородного
- Складываем решения

Таким образом, самой сложной задачей становится поиск частного решения. Для этого существует несколько методов, схожих по своей сути (*использованию неопределённых коэффициентов*).

Если правая часть уравнения имеет вид $P_m(x)e^{ax}$ и при этом $a$ не будет корнем характеристического полинома, то частное решение будет иметь вид $Q_m(x)e^{ax}$. Подставляем частное решение в исходное уравнение и приравняем коэффициенты при одинаковых степенях $x$. Получим алгебраическую систему из $m + 1$ уравнений, благодаря которой найдём коэффициенты полинома $Q_m$:

*Пример:*
$$
\frac{d^2y}{dx^2} + 4y = 2x + 3 \newline
(D^2 + 4)y = 2x + 3 \newline
P(\lambda) = \lambda^2 + 4 \newline
\lambda = \plusmn 2i \newline
y_{общ} = C_1 e^{2ix} + C_2 e^{-2ix} \newline
y_{общ} = С_1 \cos 2x + C_2 \sin 2x \newline
2x + 3 \rArr Q_1(x)e^{0x} \newline
\rArr Q_1(x) = Ax + B \newline
4Ax + 4B = 2x + 3 \newline
\begin{cases}
  4A = 2 \newline
  4B = 3 \newline
\end{cases}
B = 3/4, A = 1/2 \rArr Q_1(x) = \frac{2x + 3}{4} \newline
\rArr y = С_1 \cos 2x + C_2 \sin 2x + \frac{2x + 3}{4}
$$

**Если $a$ - корень общего решения однородного диффура степени $k$**, то частное решение примет вид $x^kQ_m(x)e^{ax}$. В остальном метод решения идентичен

*Пример:*
$$
(D^3 + D^2)y = x^2 \newline
(D^3 + D^2)y = x^2e^{0x} \newline
P(l) = l^3 + l^2 \newline
l^3 + l^2 = 0 \rArr l^2(l + 1) = 0 \rArr l_1 = l_2 = 0 = a, l_3 = -1
\rArr y_{общ} = C_1x + C_2 + C_3e^{-x} \newline
y_{частн} = x^2Q_2(x)e^{0x} = x^2(Ax^2 + Bx + C) = Ax^4 + Bx^3 + Cx^2 \newline
y^{(2)} = 12Ax^2 + 6Bx + 2C \newline
y^{(3)} = 24Ax + 6B \newline
24Ax + 6B + 12Ax^2 + 6Bx + 2C = x^2 \newline
\begin{cases}
  12A = 1 \newline
  24A + 6B = 0 \newline
  6B + 2C = 0 \newline
\end{cases}
A = 1/12 \rArr B = -1/3 \rArr C = 1 \newline
\rArr y_{частн} = \frac{Ax^4}{12} - \frac{x^3}{3} + x^2 \newline
\rArr y = C_1x + C_2 + C_3e^{-x} + \frac{Ax^4}{12} - \frac{x^3}{3} + x^2
$$

Если правая часть представлена в виде $e^{ax}[P_m(x)\cos bx + Q_l(x)\sin bx]$ и $a + bi$ **не является корнем общего решения однородного диффура**, то частное решение представима в виде $e^{ax}[R_p(x)\cos bx + T_p(x)\sin bx], p = \max(m,l)$. Соответственно, если $a + bi$ **является корнем степени $k$**, мы домножаем частное решение на $x^k$. В остальном метод решения идентичен.

**Если правая часть представима в виде суммы нескольких функций, то её частное решение будет суммой частных решений всех этих функций.** Бывает полезно в случаях, когда у нас смешаны вещественные и комплексные числа, или разные степени у экспоненты.

## Линейные уравнения с переменными коэффициентами
Вместо констант $a_i$ у нас теперь идут функции $a_i(x)$

$L(y) \equiv y^{(n)} + a_1(x)y^{(n-1)} + ... + a_{n-1}(x)y' + a_n(x)y$ - **линейный дифференциальный оператор порядка $n$** (*Именно так выглядит левая часть лин. диффура с переменными коэффициентами*). Линейный оператор обладает свойством линейности (*вот это поворот!*)

*Освежим память* функции $y_n(x)$ будут линейно независимы, если их линейная комбинация равняется нулю только при умножении функций на ноль ($\forall x \in <a,b> : \sum a_iy_i(x) = 0, a_1 = ... = a_n = 0$). Линейно независимыми будут системы $\{x^i\}$ и $\{e^{\lambda_ix}| \forall i\not=j : \lambda_i \not = \lambda_j \}$. Система $\cos^2 x, \sin^2 x, 1$ будет линейно зависима.

**Определитель Вронского** Определитель матрица для системы из $n$ функций, где $w_{ij} = y_j^{(i-1)}(x)$ - производная j-й функции порядка $i-1$ (то каждая строка - производные всех функций порядка от 1 до $n$).

**Т.** Если система функций линейно зависима в интервале, то определитель Вронского будет равняться нулю в каждой точке этого интервала. *Доказывается через факт линейной зависимости и выражение функции $y_n$ через остальные функции, а затем её дифференцирование $n-1$ раз, чтобы получить последний столбец матрицы, который будет представлять собой линейную комбинацию предыдущих, из **чего и следует равенство определителя матрицы нулю***.

**Обратное теореме утверждение не верно** - не любая система функций с нулевым определителем Вронского будет линейно зависима.

**Т.** Пусть функции $y_i(x)$ линейно независимы на $(a, b)$ и каждая из них является решением диффура. $L(y) = 0$, тогда определитель Вронского для этих функций не будет равняться нулю ни в одной из точек $(a,b)$. *Доказывается от противного через предположение, что $y_i(x)$ - линейно независимые решения $L(y) = 0$ и при этом в какой-то из точек $(a, b)$ $W = 0$. Через решение СЛАУ и условие о единственности решения задачи Коши приходим к выводу, что система функций должна быть линейно зависимой $\rArr$ **доказано через противоречие***

**С. 1** Если для системы решений $L(y) = 0$ $\{y_i\}$ определитель Вронского равен нулю в одной точке, то он будет равен нулю во всех. *Доказывается выводом от противного прошлой теоремы и позапрошлой теоремой*

**С. 2** Если определитель Вронского системы решений $L(y) = 0$ $\{y_i\}$ отличен от нуля хотя бы в одной точке, то он отличен от нуля во всех точках. *Доказательство столь же тривиально, как у прошлого следствия*

**С. 3** Для линейной независимости системы решений $L(y) = 0$ $\{y_i\}$ **необходимо и достаточно**, чтобы определитель Вронского этой системы был отличен от нуля хотя бы в одной из точек области решения.

**Т. О формуле Остроградского-Лиувилля.** Если $y_i$ - решения $L(y) = 0$, тогда 

$$
\forall x,x_0 \in (a,b) : W(x) = W(x)e^{-\int_{x_0}^xa_1(s)ds}
$$
**Существенно, что тут $a_1(x)$ именно по нашей нотации, то есть мы обязательно берём функцию при $y^{(n-1)}$, $a_0(x)$ строго $= 1$**

*Доказывается через тот факт, что производная определителя Вронского будет равняться сумме $n$ определителей, где строки поочерёдно заменяются на свои производные (то есть для всех строк кроме последней определители будут нулевыми). Затем через умножение $n-1$ строчек определителя на $a_n, ..., a_2$ и факт того, что любое $y_i$ - решение, получаем (не особо понимаю, как), что $W'(x) = -a_1(x)W(x)$, а затем домножаем выражение на $e^{\int_{x_0}^xa_1(s)ds}$ и путём "нехитрых" преобразований получаем искомую формулу*

**Фундаментальная система решений** - совокупность решений $L(y) = 0$, определённых и линейно независимых на $(a, b)$.

**Т.** Для любой $L(y)$ с непрерывными на $(a,b)$ коэффициентами $a_i(x)$ найдётся ФСР. *Доказывается подбором таких решений $y_1(x), ..., y_n(x)$, что их определитель Вронского будет определителем диагональной матрицы, то есть будет равняться единице (в лекции почему-то считается очевидным, что матрица будет такой, хотя мне это таковым не кажется...)*

**Т.** Любая линейная комбинация ФСР также будет решением $L(y) = 0$

**Т.** В однородном $L(y)$ не может быть больше $n$ линейно независимых решений.

# 23.10.10 - семинар
## Понижение порядка
Методы понижения степени:
1. Пусть в уравнении участвуют лишь производные от $m$ до $n$ и $m > 0$, то мы производим замену $u = y^{(m)}$. Решив диффур относительно $u$, производим обратную замену и интегрируем $m$ раз. (*очевидно, в решении будет $m$ произвольных постоянных, во всех последующих видах также будет больше одной константы*)
2. Уравнение, разрешённое относительно некоторой функции, явно не зависящей от $x$: $f(y, y', ..., y^{(n)}) = 0$. Здесь мы производим замену $y' = p(y) \rArr y'' = pp'$. **$p$ дифференцируется по $y$, которая является для $p$ аргументом.**. Найдя $p$, решаем диффур с разделяющимися переменными
3. Пусть уравнение однородно по группе всех производных, тогда проводим похожую замену на замену общей однородности: $y'(x) = y(x)z(x) \rArr y'' = zy' + yz' \rArr \frac{y''}{y} = z^2 + z'$
4. Выделение точной производной, *например*, мы замечаем $(y^3)' = 3y^2y'$

Методы 2 и 3 понижают порядок на единицу. Остальные могут и больше.

# 23.10.12 - лекция
## Неоднородные линейные диффуры с переменными коэффициентами
Очевидно, что вместо $L(y) = 0$ будет $L(y) = f(x)$. Как и для однородных, общее решение = частное решение + ФСР (*следует из линейности ФСР*)

Обратное утверждение также верно: если мы знаем решение неоднородного линейного диффура, то найдутся такие константы $C_1,...,C_n$, что они выделят из общего решения частное решение и ФСР.

Если была найдена ФСР, то частное НЛДУПК (*ужасная аббревиатура, но я не хочу больше писать "неоднородное линейное дифференциальное уравнение с переменными коэффициентами"*) можно найти методом вариации постоянных:
- Представим частное решение через ФСР: $y_0(x) = C_1(x)y_1(x) + ... + C_n(x)y_n(x)$. Теперь наша цель - найти $C_i(x)$
- Дифференцируем частное решение: $y_0'(x) = C_1(x)y_1'(x) + ... + C_n(x)y_n'(x) + C_1'(x)y_1(x) + ... + C_n'(x)y_n(x)$ и потребуем, чтобы $C_1'(x)y_1(x) + ... + C_n'(x)y_n(x) = 0$. Вновь дифференцируем частное решение с тем же требованием. Продолжаем до производной порядка $n-1$
- Получаем следующие системы:
$$
\begin{cases}
  C_1'(x)y_1(x) + ... + C_n'(x)y_n(x) = 0 \newline
  C_1'(x)y_1'(x) + ... + C_n'(x)y_n'(x) = 0 \newline
  ... \newline
  C_1'(x)y_1^{(n-2)}(x) + ... + C_n'(x)y_n^{(n-2)}(x) = 0 \newline
\end{cases}
$$
$$
\begin{cases}
  y_0'(x) = C_1(x)y_1'(x) + ... + C_n(x)y_n'(x) \newline
  ... \newline
  y_0^{(n-1)}(x) = C_1(x)y_1^{(n-1)}(x) + ... + C_n(x)y_n^{(n-1)}(x)
\end{cases}
$$
- Из последней строчки второй системы дифференцированием получаем $n$-ю производную: $y_0^{(n)}(x) = C_1(x)y_1^{(n)}(x) + ... + C_n(x)y_n^{(n)}(x) + C_1'(x)y_1^{(n-1)}(x) + ... + C_n'(x)y_n^{(n-1)}(x)$ (**тут не обрезаем часть с производными от $C_i$-х!**)
- Подставляем вычисленные производные в исходное уравнение и, за счёт того, что $y_1,...,y_n$ - решения однородной версии, получаем в итоге просто $C_1'(x)y_1^{(n-1)}(x) + ... + C_n'(x)y_n^{(n-1)}(x) = f(x)$, которое добавляется в первую систему. Получаем:
$$
\begin{cases}
  C_1'(x)y_1(x) + ... + C_n'(x)y_n(x) = 0 \newline
  C_1'(x)y_1'(x) + ... + C_n'(x)y_n'(x) = 0 \newline
  ... \newline
  C_1'(x)y_1^{(n-2)}(x) + ... + C_n'(x)y_n^{(n-2)}(x) = 0 \newline
  C_1'(x)y_1^{(n-1)}(x) + ... + C_n'(x)y_n^{(n-1)}(x) = f(x) \newline
\end{cases}
$$
- Определитель этой системы будет определителем Вронского от линейно независимых решений уравнения $y_1,...,y_n$, а значит система имеет единственное решение, которое может быть найдено даже методом Крамера
- Решим систему, получаем $C_1'(x) = g_1(x), ..., C_n'(x) = g_n(x)$, интегрируем их и наконец **находим частное решение** $y_0(x)$

*Практическое применение рассмотрим на семинарах (надеюсь)*

Если правая часть НДУПК $f(x) = f_1(x) + ... + f_n(x)$, то решением будет сумма решений НДУПК с правой частью только из $f_i(x)$

## (Не)Зависимость системы функций без определителя Вронского
**Определитель Грама** $\Gamma(y_1,...,y_m) = |A|$, где $a_ij = \int_a^b y_j(t)y_i(t)dt$.

Система функций линейно независима на $(a, b) \hArr \Gamma(\vec{y}) \ne 0$

**Л.** Если система функций линейно независима на интервале, то и любая её подсистема будет линейно независима. (*Доказательство: если предположить обратное, то найдутся такие $1 \le j \le k : a_j \ne 0 : a_1y_1 + ... + a_ky_k = 0$, но тогда при $k < i \le n : a_i = 0 : a_1y_1 + ... + a_ky_k + ... + a_ny_n = 0$, а это противоречит условию*)

**Л.** Если какая-либо из подсистем системы функций линейно зависима, то и вся система будет линейно зависима (*доказывается от противного банально по прошлой лемме*)

**Л.** Если функции линейно независимы на $(a,b) \rArr$ они линейно независимы на любом $(c,d)$ таком, что $(a,b) \subseteq (c,d)$ (*Следует из факта, что линейно зависимые функции должны быть зависимы на всём интервале, а на его части $(a, b)$ они линейно независимы по условию)

По линейно независимой системе непрерывно дифференцируемых $n$ раз функций $y_1, ..., y_n$ с определителем Вронского не равным нулю можно найти единственное однородное уравнение, решив систему вида:
$$
\begin{cases}
  y_1^{(n)} + p_1(x)y_1^{(n-1)} + ... + p_{n-1}(x)y_1' + p_n(x) y_1 = 0 \newline
  y_2^{(n)} + p_1(x)y_2^{(n-1)} + ... + p_{n-1}(x)y_2' + p_n(x) y_2 = 0 \newline
  ... \newline
  y_n^{(n)} + p_1(x)y_n^{(n-1)} + ... + p_{n-1}(x)y_n' + p_n(x) y_n = 0 \newline
\end{cases}
$$
*Решаться она будет без проблем методом Гаусса или Крамера (т.к. её определитель - это в точности определитель Вронского)*

**Л.** Если функции $y_1, ..., y_m$ - линейно независимые на $(a, b)$ решения НДУПК с $n \ge m$, то система решений будет также линейно независима на любом вложенном интервале/ (*Доказывается также от противного через тот факт, что если система зависима, то на любом её интервале найдутся ненулевые коэффициенты, при которых она будет равна нулю, а значит $y_0(x_0) = y_0'(x_0) = ... = y_0^{(n)} = 0$ - решение задачи Коши и при том единственное. Из этих утверждений приходим к выводу, что система от вложенного интервала никак не может быть зависимой при условии линейной независимости системы от бОльшего интервала*)

## Методы решения ЛДУ с переменными коэффициентами
**Общего метода решений ЛДУПК не существует** - *расходимся (эх, если бы...)* - существует лишь ряд методов, позволяющих упростить заданное уравнение и получить решение.

Один из таких методов - замена переменных, позволяющая перейти от уравнения с переменными коэффами к уравнению с постоянными.

Производим замену $x$ на $t = \xi(x)$ и $y(x)$ на $u(t) = y(\xi^{-1}(t))$. Далее пересчитываем производные и переходим к новому диффуру - уже с постоянными коэффициентами.

### Уравнение Эйлера
Общий вид:
$$
x^ny^{(n)} + a_1x^{n-1}y^{(n-1)} + ... + a_{n-1}xy' + a_ny = f(x)
$$
- $x = e^t, x > 0 \rArr t = \ln x$
- $x = -e^t, x < 0 \rArr t = \ln (-x)$

*Пример возьму с семинара*

Из достаточно страшных вычислений выводится общая формула для характеристического полинома уравнения Эйлера:
$$
p(\lambda) = \lambda(\lambda - 1)(\lambda - 2)...(\lambda - n + 1) + ... + a_{n-2}\lambda(\lambda - 1) + a_{n-1}\lambda + a_n
$$
**Найденные через него корни дадут нам ответы в виде суммы $Ce^{\lambda t}$, то есть далее мы должны произвести замену $t = \ln x$ и получить $Cx^{\lambda}$**

### Уравнение Лагранжа
Общий вид:
$$
(ax + b)^ny^{(n)} + a_1(ax + b)^{n-1}y^{(n-1)} + ... a_{n-1}(ax + b)y' + a_ny = f(x)
$$
*Замена тоже достаточно похожая:* $ax + b = e^t, x > 0 \rArr t = \ln |ax + b|$

Также частные решения уравнения Эйлера часто удобно искать после замены $y = x^{\lambda}$, где $\lambda$ - неизвестная постоянная

*Пример:*
$$
x^2y'' - 4xy' + 6y = 0 \newline
\rArr p(\lambda) = \lambda(\lambda - 1) - 4\lambda + 6 \newline
\lambda(\lambda - 1) - 4\lambda + 6 = 0 \newline
\lambda^2 - 5\lambda + 6 = 0 \newline
(\lambda - 2)(\lambda - 3) = 0 \newline
\rArr y = C_1e^{2t} + C_2e^{3t} \rArr y = C_1x^2 + C_2x^3 \newline
$$
*Альтернативный вариант:*
$$
x^2y'' - 4xy' + 6y = 0 \newline
y = x^{\lambda} \rArr y' = \lambda x^{\lambda - 1} \rArr y'' = \lambda(\lambda - 1) x^{\lambda - 2} \newline
x^2\lambda(\lambda - 1) x^{\lambda - 2} - 4x\lambda x^{\lambda - 1} + 6x^{\lambda} = 0 \newline
x^{\lambda}[x^2\lambda(\lambda - 1) x^{-2} - 4x\lambda x^{-1} + 6] = 0 \newline
x^{\lambda}[\lambda(\lambda - 1) - 4\lambda + 6] = 0 \newline
x^{\lambda} > 0, \space\space\space
\lambda(\lambda - 1) - 4\lambda + 6 = 0 \newline
... аналогично ...
$$

$$
x \in (-1, 1), n \in \N \newline
(1 - x^2)y'' - xy' + n^2y = 0 \newline

## Устойчивость
Для системы уравнений $y' = f(t,y)$ и задачи Коши для $t_0, y_0$. Будем говорить, что решение задачи - $\tilde{y_0}(t)$ **устойчиво (по Ляпунову)**, если:

2. Для той же системы будет задача Коши $(t_0, y_1)$ с решением $\tilde{y_1}(t)$:

## Производная комплексной функции
Комплексная функция $f(z) =  = w$ **дифференцируема в некоторой точке из области $D$**, если $\exist$ конечный $\lim_{\Delta z \rarr 0} \frac{f(z + \Delta z) - f(z)}{\Delta z} = \lim_{\Delta z \rarr 0} \frac{\Delta w}{\Delta z}$. Такой предел будет называться **производной комплексной функции** и будет обозначаться штрихом.

Ввиду двумерности комплексных чисел нам безразлично, с какой стороны мы стремимся к $z$

Представим $\Delta z = \Delta x + i\Delta y$, тогда можем определить такие производные:

- *аналогичным образом (почти) определяется производная при $\Delta y = 0, \Delta x \rarr 0$ (но лучше добавить фотку)*

- $\frac{\delta u}{\delta x} = \frac{\delta v}{\delta y}$
- $\frac{\delta v}{\delta x} = -\frac{\delta u}{\delta y}$

*Для примера можно легко проверить дифференцируемость функции $f(z) = z$*

## Интегральная формула Коши
Пусть функция $f(z)$ аналитична в области $D$, интегрируема на кривой $\Gamma$ (контуре области $D$) и непрерывна в замыкании $\Gamma$, тогда:
$$
\frac{1}{2i\pi}\int_{\Gamma} \frac{f(w)dw}{w - z} = \begin{cases}
  f(z), z \in D \\
  0, z \in \Complex - \overline{D}
\end{cases}
$$
*Доказательство:*
- Для второго случая функция $\frac{f(w)dw}{w - z}, \forall w \in \Gamma$ будет аналитичной, а значит по теореме Коши интеграл по Гамме будет равняться нулю
- Для первого случая внутри области проводим круг радиуса $\delta$ с центром в точке $S$. $D - c(S, \delta) = D_1$ - замыкание круга. Это замыкания будет обладать двумя несвязными границами. В области $D_1$ функция $\frac{f(w)dw}{w - z}$ аналитична и непрерывна в замыкании $D_1$. Будем интегрировать Гамму против часовой стрелки, а внутреннюю границу $D_1$ - против. Значит, мы можем записать нечто вида:
$$
\int_{\Gamma} \frac{f(w)dw}{w - z} = \int_{|w - z| < \delta} \frac{f(w)dw}{w - z} \\
$$
Проводим расчёты (*добавить фотку*) и получаем, что:
$$
\lim_{\delta \rarr 0} \int_{|w - z| < \delta} \frac{f(w)dw}{w - z} = 2i\pi f(z) \\
\rArr \int_{\Gamma} \frac{f(w)dw}{w - z} = 2i\pi f(z)

## Изучение особых точек функции
Нужно для счёта интегралов

$z$ - собоая точка, если $f(z)$ не определена. Пример: $1/z$ при $z = 0$

**Изолированная особая точка** - в какой-то окрестности нет других особых точек. Отсюда следует и определение **неизолированной** - невозможно подобрать окрестность, в которой нет других особых точек.

Пример: $\frac{1}{\sin z}$, здесь $\pi k$ - изолированные особые точки, $\infty$ - неизолированная

Работаем только с **изолированными особыми точками**, которые делятся на:
1. Устранимые: $\lim_{z \rarr z_0} f(z) < \infty$ - главная часть ряда Лорана отсутствует
2. Полюс порядка $k$: $\lim_{z \rarr z_0} f(z) = \infty$ либо в главной части $k$ слагаемых (можно представить в виде $\frac{1}{(z - z_0)^k}\phi(z)$, где $\phi(z)$ - аналитичная)
3. Существенно-особая точка: либо предела не существует, либо главная часть имеет бесконечное число слагаемых

**Критерий полюса**:
1. $z_0$ - полюс порядка $k$, если $\lim_{z \rarr z_0}(z - z_0)^{k-1}f(z) = \infty$, но $\lim_{z \rarr z_0}(z - z_0)^kf(z) = w_0$ - конечное комплексное число
2. Аналитическая функция $f(z)$, $z_0$ - нуль порядка $k$, если $f(z_0) = f'(z_0) = ... = f^{{k-1}}(z_0) = 0$ и $f^{(k)} \ne 0$

При разложении на ряд Лорана в бублике главная часть будет с отрицательными степенями, а при разложении во внейшней части (бублик до бесконечности), тогда в главной части будут положительные степени

**Вычет** функции $f(z)$ в точке $z_0$:
$$
Res_{z = z_0} f(z) = \frac{1}{2i\pi}\int_{\gamma} f(z)dz 
$$

$$
**ВАЖНЫЙ ПРИМЕР**

**Формулы вычетов:**
1. Для конечной устранимой точки выечт будет 0
2. Для полюса порядка $k$: $Res_{z = z_0}f = \frac{1}{(k-1)!}\lim_{z\rarr z_0}\frac{d^{k-1}}{dz^{k-1}}((z-z_0)^kf(z))$
3. Для просто полюса ($k = 1$): работаем с аналитичным числителем $\phi$ и знаменателем $\xi$, производная котрого не равна нулю: $Res_{z = z_0}f = \frac{\phi(z_0)}{\xi'(z_0)}$

5. Для бесконечной ОТ **ЛЮБОГО ТИПА** вычет равен $-C_{-1}$

**Основная теорема о вычетах**. $f(z)$ аналитична в $D$ за исключением **конечного** числа изолированных особых точек, тогда:
$$
\int_D f(z)dz = 2i\pi \sum_{i \in I} Res_{z = z_i}  f
$$
$I$ - индексы особых точек
